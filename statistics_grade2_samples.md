## 1. 記述統計分野

### Q1. 相関係数・共分散の計算
あるクラスの10人の生徒に対して行われた数学テストの得点 $X$ 点と物理テストの得点 $Y$ 点のデータがある。
それぞれの平均値は $\bar{x} = 60$, $\bar{y} = 55$ であり、分散は $s_x^2 = 100$, $s_y^2 = 64$ であった。
また、得点の積の平均 $\overline{xy}$ は $3340$ であった。
このとき、数学と物理の得点間のピアソンの相関係数 $r$ として最も近いものを以下の選択肢から選べ。

1.  $0.35$
2.  $0.50$
3.  $0.65$
4.  $0.75$
5.  $0.85$

*   **解答**: **2. $0.50$**
*   **解説**:
    1.  **共分散 $s_{xy}$ を求める**
        $$s_{xy} = \overline{xy} - \bar{x}\bar{y} = 3340 - (60 \times 55) = 40$$
    2.  **標準偏差 $s_x, s_y$ を求める**
        $$s_x = \sqrt{100} = 10, \quad s_y = \sqrt{64} = 8$$
    3.  **相関係数 $r$ を求める**
        $$r = \frac{s_{xy}}{s_x s_y} = \frac{40}{10 \times 8} = 0.50$$

### Q2. 変動係数（CV）
製品A群の重量データの平均は $200\text{g}$、標準偏差は $10\text{g}$ である。
製品B群の重量データの平均は $1000\text{g}$、標準偏差は $40\text{g}$ である。
それぞれのデータの相対的なばらつきを示す「変動係数（CV）」の組み合わせ、およびばらつきが相対的に大きい方の製品の判定として正しいものを以下の選択肢から選べ。

1.  $CV_A = 0.05$, $CV_B = 0.04$ / 製品Aの方が相対的なばらつきが大きい
2.  $CV_A = 0.05$, $CV_B = 0.04$ / 製品Bの方が相対的なばらつきが大きい
3.  $CV_A = 20.0$, $CV_B = 25.0$ / 製品Bの方が相対的なばらつきが大きい
4.  $CV_A = 0.08$, $CV_B = 0.05$ / 製品Aの方が相対的なばらつきが大きい
5.  $CV_A = 0.04$, $CV_B = 0.05$ / 製品Bの方が相対的なばらつきが大きい

*   **解答**: **1. $CV_A = 0.05$, $CV_B = 0.04$ / 製品Aの方が相対的なばらつきが大きい**
*   **解説**:
    1.  **変動係数の定義**
        変動係数（Coefficient of Variation）は、標準偏差を平均値で割った値です。単位の異なるデータや、平均値が大きく異なるデータのばらつきを相対的に比較する際に用いられます。
        $$CV = \frac{\text{標準偏差}}{\text{平均値}}$$
    2.  **各製品のCV計算**
        - 製品A: $CV_A = \frac{10}{200} = 0.05 \quad (5\%)$
        - 製品B: $CV_B = \frac{40}{1000} = 0.04 \quad (4\%)$
    3.  **判定**
        $CV_A > CV_B$ であるため、製品Aの方が相対的なばらつきが大きいです。

### Q3. 箱ひげ図と四分位数
ある商品の1日の販売個数に関して、30日間のデータをまとめたところ、第1四分位数 $Q_1 = 150$ 個、中央値 $Q_2 = 180$ 個、第3四分位数 $Q_3 = 220$ 個であった。
このデータに関する記述として、**最も適切なもの**を以下の選択肢から選べ。

1.  販売個数の四分位範囲（IQR）は $30$ 個である。
2.  販売個数が $180$ 個以上 $220$ 個以下であった日は、少なくとも15日ある。
3.  販売個数が $150$ 個未満であった日は、全体の約 $25\%$ である。
4.  データの範囲（レンジ）は $70$ 個である。
5.  販売個数の平均値は必ず $180$ 個になる。

*   **解答**: **3. 販売個数が $150$ 個未満であった日は、全体の約 $25\%$ である。**
*   **解説**:
    1.  **四分位数の定義**
        データを小さい順に並べたときに、 $25\%$ 地点にあるのが第1四分位数（$Q_1$）、 $50\%$ 地点にあるのが中央値（$Q_2$）、 $75\%$ 地点にあるのが第3四分位数（$Q_3$）です。
    2.  **各選択肢の吟味**
        - 選択肢1: 四分位範囲（IQR）は $Q_3 - Q_1 = 220 - 150 = 70$ です。よって誤り。
        - 選択肢2: $180$ 個 ($Q_2$) から $220$ 個 ($Q_3$) の間には、全データの約 $25\%$ (30日中の約7〜8日) が存在します。よって誤り。
        - 選択肢3: $Q_1 = 150$ 個未満のデータは全体の約 $25\%$ を占めます。これが適切です。
        - 選択肢4: データの範囲（レンジ）は「最大値 － 最小値」であり、四分位数だけでは求まりません。よって誤り。
        - 選択肢5: 平均値はデータの分布の非対称性によって中央値とズレるため、必ずしも $180$ 個になるとは限りません。よって誤り。

### Q4. ローレンツ曲線とジニ係数
ある地域における5世帯の所得（万円）が、それぞれ $200, 300, 500, 1000, 2000$ であった。
この所得データから描かれるローレンツ曲線、および所得分配の不平等さを表す「ジニ係数」に関する記述として、**最も不適切なもの**を以下の選択肢から選べ。

1.  世帯全体の所得の合計は $4000$ 万円であり、所得が最も低い世帯（200万円）のシェアは $5\%$ である。
2.  所得の低い順に世帯を並べたとき、累積世帯数割合（横軸）と累積所得割合（縦軸）をプロットして結んだ曲線をローレンツ曲線と呼ぶ。
3.  すべての世帯の所得が完全に等しいとき、ローレンツ曲線は対角線（均等分布線）に一致し、ジニ係数は $0$ となる。
4.  1世帯のみが所得を独占し、他がすべて $0$ である極端な不平等状態では、ジニ係数は $1$ に近づく。
5.  ローレンツ曲線が対角線から離れて右下に膨らむほど、所得分配は平等になっておりジニ係数は小さくなる。

*   **解答**: **5. ローレンツ曲線が対角線から離れて右下に膨らむほど、所得分配は平等になっておりジニ係数は小さくなる。**
*   **解説**:
    1.  **ローレンツ曲線とジニ係数の関係**
        ローレンツ曲線が均等分布線（対角線）から離れて右下に膨らむ（弓なりになる）ほど、分配の不平等度が高くなっていることを示します。
    2.  **ジニ係数**
        均等分布線とローレンツ曲線で囲まれた面積を $A$、ローレンツ曲線の下側の面積を $B$ とすると、ジニ係数は $\frac{A}{A+B}$ で定義されます。したがって、不平等になり対角線から離れるほど面積 $A$ が大きくなり、ジニ係数は $1$ に近づきます。
    3.  **選択肢5の誤り**
        対角線から離れるほど「不平等」になり、ジニ係数は「大きく」なります。よって選択肢5が不適切です。

### Q5. 代表値の関係（平均値・中央値・最頻値）
日本の所得分布のように、一部の高額所得者によって右側に長い裾を持つ（右に歪んだ、あるいは正の歪度を持つ）非対称な分布について、平均値（Mean）、中央値（Median）、最頻値（Mode）の3つの大小関係として最も一般的なものを以下の選択肢から選べ。

1.  平均値 $<$ 中央値 $<$ 最頻値
2.  最頻値 $<$ 中央値 $<$ 平均値
3.  中央値 $<$ 最頻値 $<$ 平均値
4.  平均値 $=$ 中央値 $=$ 最頻値
5.  最頻値 $<$ 平均値 $<$ 中央値

*   **解答**: **2. 最頻値 $<$ 中央値 $<$ 平均値**
*   **解説**:
    1.  **非対称分布と代表値の関係**
        - 左右対称の山型分布（正規分布など）では、平均値 ＝ 中央値 ＝ 最頻値 となります。
        - **右裾が長い分布（右に歪んだ分布）**では、右側の極端に大きい値（外れ値）によって**平均値**が右側に大きく引っ張られます。
        - 一方、**最頻値**は最も度数が高い場所（山の頂上）にとどまります。
        - **中央値**は順位の真ん中の値であるため、平均値と最頻値の中間に位置します。
    2.  **大小関係**
        山の頂上（最頻値）が最も左にあり、中央値がその右、平均値がさらに右に引っ張られるため、 `最頻値 < 中央値 < 平均値` の関係になります。

## 2. 確率の基礎分野

### Q6. ベイズの定理（条件付き確率）
ある感染症のスクリーニング検査について以下の情報が得られている。
- 実際に感染している人が検査を受けて「陽性」と判定される確率（感度）は $90\%$ である。
- 感染していない人が検査を受けて「陰性」と判定される確率（特異度）は $95\%$ である。
- この地域全体において、その感染症に実際に感染している人の割合（有病率）は $2\%$ である。

この検査を受けたある人が「陽性」と判定されたとき、この人が「実際に感染している確率」として最も近いものを以下の選択肢から選べ。

1.  $0.02$
2.  $0.18$
3.  $0.27$
4.  $0.64$
5.  $0.90$

*   **解答**: **3. $0.27$**
*   **解説**:
    1.  **事象の整理**
        - $D$: 実際に感染している, \ $P(D) = 0.02 \implies P(D^c) = 0.98$
        - $+$, $-$: 検査陽性、陰性
        - $P(+ | D) = 0.90$ （感度）
        - $P(- | D^c) = 0.95 \implies P(+ | D^c) = 0.05$ （偽陽性率）
    2.  **ベイズの定理の適用**
        $$P(D | +) = \frac{P(+ | D)P(D)}{P(+ | D)P(D) + P(+ | D^c)P(D^c)}$$
        - 分子（感染していて陽性）: $0.90 \times 0.02 = 0.018$
        - 分母（全体の陽性確率）: $0.018 + (0.05 \times 0.98) = 0.018 + 0.049 = 0.067$
        - 事後確率: $P(D | +) = \frac{0.018}{0.067} \approx 0.2687 \quad (26.9\%)$

### Q7. 確率の加法定理・乗法定理・余事象
当たりくじが $3$ 本、はずれくじが $7$ 本の合計 $10$ 本が入った箱がある。この箱から同時に $3$ 本のくじを引くとき、「少なくとも $1$ 本の当たりくじを引く確率」として正しいものを以下の選択肢から選べ。

1.  $\frac{7}{24}$
2.  $\frac{17}{24}$
3.  $\frac{3}{10}$
4.  $\frac{21}{40}$
5.  $\frac{19}{40}$

*   **解答**: **2. $\frac{17}{24}$**
*   **解説**:
    1.  **余事象の考え方**
        「少なくとも1本当たる」という事象の余事象は、「3本ともすべてはずれる」です。
    2.  **余事象の確率 $P(A^c)$ の計算**
        10本から3本選ぶすべての組み合わせ数は、
        $${}_{10}C_3 = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120 \text{通り}$$
        はずれくじ7本から3本選ぶ組み合わせ数は、
        $${}_7C_3 = \frac{7 \times 6 \times 5}{3 \times 2 \times 1} = 35 \text{通り}$$
        したがって、3本ともはずれる確率は、
        $$P(A^c) = \frac{35}{120} = \frac{7}{24}$$
    3.  **求める確率 $P(A)$ の計算**
        余事象の確率を1から引きます。
        $$P(A) = 1 - P(A^c) = 1 - \frac{7}{24} = \frac{17}{24} \approx 0.708$$

### Q8. 確率変数の線形結合の期待値と分散
互いに独立な確率変数 $X$ と $Y$ がある。 $X$ の期待値は $E[X] = 4$、分散は $V[X] = 2$ であり、 $Y$ の期待値は $E[Y] = 3$、分散は $V[Y] = 1$ である。
このとき、新しい確率変数 $W = 2X - 3Y$ を定義する。 $W$ の期待値 $E[W]$ および分散 $V[W]$ の組み合わせとして正しいものを以下の選択肢から選べ。

1.  $E[W] = -1$, $V[W] = 17$
2.  $E[W] = -1$, $V[W] = -1$
3.  $E[W] = -1$, $V[W] = 5$
4.  $E[W] = 17$, $V[W] = 17$
5.  $E[W] = 1$, $V[W] = 11$

*   **解答**: **1. $E[W] = -1$, $V[W] = 17$**
*   **解説**:
    1.  **期待値の性質（線形性）**
        期待値の計算において、変数間の独立性は不要です。
        $$E[W] = E[2X - 3Y] = 2E[X] - 3E[Y] = 2(4) - 3(3) = 8 - 9 = -1$$
    2.  **分散の性質**
        $X$ と $Y$ は**独立**であるため、共分散 $Cov(X, Y) = 0$ となります。
        定数倍の分散は、定数の2乗倍になります（$V[aX] = a^2 V[X]$）。
        $$V[W] = V[2X - 3Y] = V[2X] + V[-3Y] = 2^2 V[X] + (-3)^2 V[Y] = 4V[X] + 9V[Y]$$
        与えられた値を代入します。
        $$V[W] = 4(2) + 9(1) = 8 + 9 = 17$$

## 3. 確率分布分野

### Q9. 二項分布の正規近似
ある製品の不良品発生率は $10\%$ である。この製品を無作為に $400$ 個抽出したとき、不良品の個数 $X$ が $46$ 個以上となる確率として最も近いものを以下の選択肢から選べ。
ただし、抽出数 $n=400$ は十分に大きく、二項分布は正規分布で近似（連続修正は考慮しない）できるものとする。
また、標準正規分布 $Z \sim N(0, 1)$ において、
$P(Z \geq 1.0) \approx 0.1587$, $P(Z \geq 1.5) \approx 0.0668$, $P(Z \geq 2.0) \approx 0.0228$ とする。

1.  $0.0228$
2.  $0.0668$
3.  $0.1587$
4.  $0.3413$
5.  $0.4332$

*   **解答**: **3. $0.1587$**
*   **解説**:
    1.  **二項分布のパラメータ**
        不良品数 $X \sim B(n, p)$ （$n = 400, \ p = 0.1$）
    2.  **平均と分散**
        - 平均: $\mu = np = 400 \times 0.1 = 40$
        - 分散: $\sigma^2 = np(1-p) = 400 \times 0.1 \times 0.9 = 36 \implies \sigma = 6$
    3.  **標準化**
        $X$ は近似的に $N(40, 6^2)$ に従うため、 $X = 46$ を標準化します。
        $$Z = \frac{X - \mu}{\sigma} = \frac{46 - 40}{6} = 1.0$$
        $$P(X \ge 46) \approx P(Z \ge 1.0) = 0.1587$$

### Q10. 離散確率分布（ポアソン分布・幾何分布）
確率分布に関する以下の2つの問いに答えよ。
(i) あるスーパーのレジに1時間あたり平均3人の顧客が到着する（顧客の到着はポアソン分布に従う）。このとき、ある1時間に顧客が「誰も来ない」確率。
(ii) 表が出る確率が $p = 0.2$ であるコインを投げ続け、初めて表が出るまでに「ちょうど3回連続で裏が出る（つまり4回目で初めて表が出る）」確率。
それぞれの確率の組み合わせとして正しいものを以下の選択肢から選べ。（必要ならば $e^{-3} \approx 0.050$ を用いよ）

1.  (i) $0.050$ , \ (ii) $0.1024$
2.  (i) $0.050$ , \ (ii) $0.0016$
3.  (i) $0.150$ , \ (ii) $0.1024$
4.  (i) $0.025$ , \ (ii) $0.0512$
5.  (i) $0.150$ , \ (ii) $0.0512$

*   **解答**: **1. (i) $0.050$ , \ (ii) $0.1024$**
*   **解説**:
    1.  **問(i) ポアソン分布の計算**
        平均発生数 $\lambda = 3$ のポアソン分布の確率関数は、
        $$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$$
        顧客が誰も来ない（$k=0$）確率は、
        $$P(X = 0) = \frac{3^0 e^{-3}}{0!} = e^{-3} \approx 0.050$$
    2.  **問(ii) 幾何分布の計算**
        裏が3回（失敗）続き、4回目で表（成功）が出る確率は、
        $$P(Y = 4) = (1 - p)^3 \times p = (0.8)^3 \times 0.2 = 0.512 \times 0.2 = 0.1024$$

### Q11. 連続確率分布（正規分布・指数分布）
連続型確率分布に関する記述として、**最も不適切なもの**を以下の選択肢から選べ。

1.  正規分布 $N(\mu, \sigma^2)$ は、平均値 $\mu$ を中心に左右対称な釣鐘型の分布であり、平均値と中央値と最頻値がすべて一致する。
2.  標準正規分布 $N(0, 1)$ に従う確率変数 $Z$ について、 $P(-1.96 \le Z \le 1.96) \approx 0.95$ である。
3.  指数分布は、ポアソン過程におけるイベントの発生時間間隔が従う分布であり、確率密度関数は $f(x) = \lambda e^{-\lambda x} \ (x \ge 0, \ \lambda > 0)$ と表される。
4.  指数分布は「無記憶性（失念性）」を持ち、それまでにどれだけ時間が経過したかに関わらず、次までの待ち時間の分布は常に一定である。
5.  パラメータ $\lambda$ の指数分布に従う確率変数の期待値は $\lambda$ であり、分散は $\lambda^2$ である。

*   **解答**: **5. パラメータ $\lambda$ の指数分布に従う確率変数の期待値は $\lambda$ であり、分散は $\lambda^2$ である。**
*   **解説**:
    1.  **指数分布の特性値の確認**
        確率密度関数 $f(x) = \lambda e^{-\lambda x}$ を持つ指数分布について：
        - 期待値: $E[X] = \frac{1}{\lambda}$
        - 分散: $V[X] = \frac{1}{\lambda^2}$
    2.  **選択肢5の誤り**
        期待値は $\lambda$ ではなく $\frac{1}{\lambda}$、分散は $\lambda^2$ ではなく $\frac{1}{\lambda^2}$ です。したがって、選択肢5が不適切です。

## 4. 標本分布分野

### Q12. 中心極限定理
ある部品の寿命時間 $X$ は、特定の確率分布に必ずしも従わないが、平均 $\mu = 100$ 時間、標準偏差 $\sigma = 20$ 時間の分布に従うことが分かっている。
この部品をランダムに $100$ 個抽出し、その標本平均を $\bar{X}$ とする。 $\bar{X}$ が $96$ 時間以下になる確率として最も近いものを以下の選択肢から選べ。
ただし、標本サイズ $n=100$ は十分に大きく、中心極限定理を適用できるものとする。
（標準正規分布 $Z \sim N(0, 1)$ について、 $P(Z \ge 2.0) \approx 0.0228$ を用いてよい）

1.  $0.0228$
2.  $0.0500$
3.  $0.1587$
4.  $0.3085$
5.  $0.4772$

*   **解答**: **1. $0.0228$**
*   **解説**:
    1.  **標本平均 $\bar{X}$ の分布**
        中心極限定理（CLT）より、元の分布が何であれ、標本サイズ $n$ が十分に大きければ、標本平均 $\bar{X}$ は近似的に平均 $\mu$、分散 $\frac{\sigma^2}{n}$ の正規分布に従います。
        $$\bar{X} \approx N\left(\mu, \frac{\sigma^2}{n}\right) = N\left(100, \frac{20^2}{100}\right) = N(100, 4)$$
        よって、標本平均の標準誤差（SE）は $\sqrt{4} = 2$ 時間です。
    2.  **標準化と確率計算**
        $\bar{X} = 96$ を標準化変数 $Z$ に変換します。
        $$Z = \frac{\bar{X} - \mu}{SE} = \frac{96 - 100}{2} = \frac{-4}{2} = -2.0$$
        $$P(\bar{X} \le 96) \approx P(Z \le -2.0) = P(Z \ge 2.0) = 0.0228$$

### Q13. 標本分散とカイ二乗分布
平均 $\mu$、分散 $\sigma^2$ の正規母集団 $N(\mu, \sigma^2)$ から、サイズ $n$ の標本 $X_1, X_2, \dots, X_n$ を無作為抽出する。
標本平均を $\bar{X}$、不偏分散を $s^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$ とするとき、統計量
$$W = \frac{(n-1)s^2}{\sigma^2}$$
が従う確率分布として正しいものを以下の選択肢から選べ。

1.  自由度 $n$ のt分布
2.  自由度 $n-1$ のt分布
3.  自由度 $n$ のカイ二乗分布
4.  自由度 $n-1$ のカイ二乗分布
5.  自由度 $(n-1, n-1)$ のF分布

*   **解答**: **4. 自由度 $n-1$ のカイ二乗分布**
*   **解説**:
    1.  **統計量 $W$ の分布**
        正規母集団からの標本について、偏差平方和を母分散で割った統計量 $W = \frac{\sum(X_i - \bar{X})^2}{\sigma^2} = \frac{(n-1)s^2}{\sigma^2}$ は、**自由度 $n-1$ のカイ二乗分布**（ $\chi^2(n-1)$ ）に従うことが知られています。
    2.  **自由度の減少**
        元の $n$ 個の偏差平方和において、標本平均 $\bar{X}$ を用いることで制約（ $\sum(X_i - \bar{X}) = 0$ ）が1つ生じるため、自由度が $n$ から $n-1$ に減少します。

### Q14. 母比率の信頼区間

2.  $[0.360, 0.440]$
3.  $[0.376, 0.424]$
4.  $[0.380, 0.420]$
5.  $[0.392, 0.408]$

*   **解答**: **1. $[0.352, 0.448]$**
*   **解説**:
    1.  **標本比率 $\hat{p}$**

### Q18. 2標本t検定（等分散の仮定）
新薬Aと従来薬Bの血圧降下効果を比較するため、それぞれ独立した被験者グループ（グループAは $n_1 = 10$ 人、グループBは $n_2 = 12$ 人）に対して投与実験を行った。
血圧降下量（mmHg）の平均値と不偏分散は以下の通りであった。
- グループA：平均値 $\bar{x}_A = 15.0$、不偏分散 $s_A^2 = 18.0$
- グループB：平均値 $\bar{x}_B = 11.0$、不偏分散 $s_B^2 = 12.0$

両群の血圧降下量の母分散は等しい（等分散）と仮定し、「新薬Aと従来薬Bの効果に差はない」という帰無仮説のもとで2標本t検定を行う。
このときの検定統計量 $t$ の値に最も近いものを以下の選択肢から選べ。

1.  $1.21$
2.  $1.85$
3.  $2.43$
4.  $3.10$
5.  $4.02$

*   **解答**: **3. $2.43$**
*   **解説**:
    1.  **プールされた不偏分散 $s_p^2$**
        $$s_p^2 = \frac{(n_1-1)s_A^2 + (n_2-1)s_B^2}{n_1 + n_2 - 2} = \frac{9 \times 18.0 + 11 \times 12.0}{10 + 12 - 2} = 14.7$$
    2.  **検定統計量 $t$ の計算**
        $$t = \frac{\bar{x}_A - \bar{x}_B}{\sqrt{s_p^2 \left( \frac{1}{n_1} + \frac{1}{n_2} \right)}} = \frac{15.0 - 11.0}{\sqrt{14.7 \left( \frac{1}{10} + \frac{1}{12} \right)}} = \frac{4.0}{\sqrt{14.7 \times 0.1833}} \approx \frac{4.0}{1.642} \approx 2.436$$

### Q19. 第一種・第二種の誤りと検出力

    4.  **F値 [ E ]**: $60 / 10 = 6.0$

### Q25. 二元配置分散分析表
2つの因子A（3水準）、因子B（2水準）について、それぞれ繰り返しのある実験を各組み合わせで $4$ 回ずつ行い、合計 $24$ 個のデータから二元配置分散分析を行った。
この分析における分散分析表の「各要因の自由度」の組み合わせ [因子Aの自由度, 因子Bの自由度, 交互作用A×Bの自由度, 残差の自由度, 全体の自由度] として正しいものを以下の選択肢から選べ。

1.  $[2, 1, 2, 18, 23]$
2.  $[3, 2, 6, 12, 23]$
3.  $[2, 1, 3, 18, 24]$
4.  $[2, 1, 2, 20, 25]$
5.  $[3, 2, 5, 12, 22]$

*   **解答**: **1. $[2, 1, 2, 18, 23]$**
*   **解説**:
    1.  **各要因の自由度**
        - 因子Aの水準数: $a=3 \implies df_A = a - 1 = 2$
        - 因子Bの水準数: $b=2 \implies df_B = b - 1 = 1$
        - 交互作用の自由度: $df_{AB} = (a-1)(b-1) = 2 \times 1 = 2$
        - 全体サンプル数: $N = 3 \times 2 \times 4 = 24 \implies df_{Total} = 24 - 1 = 23$
        - 残差の自由度: $df_{Error} = df_{Total} - df_A - df_B - df_{AB} = 23 - 2 - 1 - 2 = 18$
    2.  したがって、 $[2, 1, 2, 18, 23]$ が正解です。

### Q26. 単回帰モデルと決定係数
ある店舗における日ごとの「最高気温 $X$（℃）」と「清涼飲料水の売上数 $Y$（本）」の10日分のデータがある。
それぞれの偏差平方和および偏差共分散は以下のようであった。
- $X$ の偏差平方和： $S_{xx} = 80$
- $Y$ の偏差平方和： $S_{yy} = 500$
- $X$ と $Y$ の偏差共分散： $S_{xy} = 160$

このデータを用いて、単回帰モデル $Y = \beta_0 + \beta_1 X + u$ を最小二乗法で推定したとき、推定された回帰係数の傾き $\hat{\beta}_1$ と、決定係数 $R^2$ の組み合わせとして正しいものを以下の選択肢から選べ。

1.  $\hat{\beta}_1 = 2.0$, \ $R^2 = 0.64$
2.  $\hat{\beta}_1 = 2.0$, \ $R^2 = 0.80$
3.  $\hat{\beta}_1 = 0.5$, \ $R^2 = 0.64$
4.  $\hat{\beta}_1 = 0.5$, \ $R^2 = 0.80$
5.  $\hat{\beta}_1 = 2.0$, \ $R^2 = 0.40$

*   **解答**: **1. $\hat{\beta}_1 = 2.0$, \ $R^2 = 0.64$**
*   **解説**:
    1.  **回帰係数の傾き**: $\hat{\beta}_1 = \frac{S_{xy}}{S_{xx}} = \frac{160}{80} = 2.0$
    2.  **決定係数**: 相関係数 $r = \frac{160}{\sqrt{80 \times 500}} = \frac{160}{200} = 0.8 \implies R^2 = r^2 = 0.64$

### Q27. 重回帰分析
ある企業の売上高 $Y$ を、広告費 $X_1$ および店舗面積 $X_2$ の2つの説明変数で回帰した重回帰モデルを構築した。
サンプルサイズは $n = 23$ である。この回帰モデルの決定係数（寄与率）は $R^2 = 0.60$ であった。
このときの「自由度調整済み決定係数 $\bar{R}^2$」の値として最も近いものを以下の選択肢から選べ。

1.  $0.52$
2.  $0.54$
3.  $0.56$
4.  $0.58$
5.  $0.60$

*   **解答**: **3. $0.56$**
*   **解説**:
    1.  **自由度調整済み決定係数**
        $$\bar{R}^2 = 1 - (1 - R^2) \frac{n - 1}{n - k - 1}$$
        $n = 23$ (標本数)、 $k = 2$ (説明変数数) を代入します。
        $$\bar{R}^2 = 1 - (1 - 0.60) \times \frac{22}{20} = 1 - 0.40 \times 1.1 = 1 - 0.44 = 0.56$$

### Q28. 物価指数（ラスパイレス指数とパーシェ指数）
ある市場における2つの主要品目（品目A、品目B）の基準年（期）と比較年（期）の価格および取引数量は以下の通りであった。

| 品目 | 基準年価格 ($P_0$) | 基準年数量 ($Q_0$) | 比較年価格 ($P_t$) | 比較年数量 ($Q_t$) |
| :--- | :---: | :---: | :---: | :---: |
| **品目A** | 100円 | 10個 | 120円 | 12個 |
| **品目B** | 200円 | 5個 | 220円 | 6個 |

基準年を $100$ としたときの、比較年の「ラスパイレス物価指数 $I_L$」および「パーシェ物価指数 $I_P$」の正しい組み合わせを以下の選択肢から選べ。

1.  $I_L = 110.0$ , \ $I_P = 111.1$
2.  $I_L = 115.0$ , \ $I_P = 113.8$
3.  $I_L = 115.0$ , \ $I_P = 115.0$
4.  $I_L = 113.8$ , \ $I_P = 115.0$
5.  $I_L = 110.0$ , \ $I_P = 110.0$

*   **解答**: **3. $I_L = 115.0$ , \ $I_P = 115.0$**
*   **解説**:
    1.  **ラスパイレス式**
        $$I_L = \frac{\sum P_t Q_0}{\sum P_0 Q_0} \times 100 = \frac{(120 \times 10) + (220 \times 5)}{(100 \times 10) + (200 \times 5)} \times 100 = \frac{2300}{2000} \times 100 = 115.0$$
    2.  **パーシェ式**
        $$I_P = \frac{\sum P_t Q_t}{\sum P_0 Q_t} \times 100 = \frac{(120 \times 12) + (220 \times 6)}{(100 \times 12) + (200 \times 6)} \times 100 = \frac{1440 + 1320}{1200 + 1200} \times 100 = \frac{2760}{2400} \times 100 = 115.0$$

### Q29. 時系列分析（移動平均、自己相関、トレンド）
時系列データの分析手法に関する記述として、**最も適切なもの**を以下の選択肢から選べ。

1.  時系列データの傾向（トレンド）を滑らかに抽出するための「3期移動平均」では、時刻 $t$ における値を $Y_t$ とするとき、常に $\frac{Y_t + Y_{t+1} + Y_{t+2}}{3}$ を用いて計算し、中心化は行わない。
2.  時系列データが長期にわたって一定の方向に増加または減少する傾向を「季節変動」と呼ぶ。
3.  「自己相関関数（ACF）」とは、同一の時系列データにおいて、時刻 $t$ の値とそこから一定時間（ラグ $k$）離れた時刻 $t-k$ の値との間の相関係数を示したものである。
4.  自己相関コレログラムにおいて、すべてのラグ $k \ge 1$ で自己相関がほぼ $0$ であるデータ（ホワイトノイズ）は、過去のデータから将来の値を完全に予測できる。
5.  月次データにおいて、毎年同じ月（例えば毎年12月に売上が上がるなど）に繰り返される変動パターンを「循環変動」と呼ぶ。

*   **解答**: **3. 「自己相関関数（ACF）」とは、同一の時系列データにおいて、時刻 $t$ の値とそこから一定時間（ラグ $k$）離れた時刻 $t-k$ の値との間の相関係数を示したものである。**
*   **解説**:
    1.  長期の変動は「トレンド（傾向変動）」であり、毎年繰り返す変動は「季節変動」です。
    2.  ホワイトノイズは過去の情報から予測できません。
    3.  中心化3期移動平均は $\frac{Y_{t-1} + Y_t + Y_{t+1}}{3}$ を用います。
    4.  よって、選択肢3が正しい自己相関の定義であり適切です。

### Q30. 標本抽出法
母集団から標本を無作為に抽出する種々の方法に関する記述として、**最も不適切なもの**を以下の選択肢から選べ。

1.  母集団のリストから、最初の1個をランダムに選び、以降は一定の間隔（例えば10個おき）で機械的に抽出する方法を「系統抽出法」と呼ぶ。
2.  母集団をいくつかのグループ（層）に分割し、各層からその大きさに比例する（または一定の）サンプルサイズを無作為に抽出する方法を「層化無作為抽出法」と呼ぶ。
3.  母集団を多数の小グループ（集落・クラスター）に分け、そこからいくつかの集落をランダムに選び、選ばれた集落の構成員全員を調査する方法を「クラスター抽出法（集落抽出法）」と呼ぶ。
4.  「多段抽出法」では、まず第1次抽出単位（例えば都道府県）を抽出し、次に第2次抽出単位（市区町村）、さらに第3次抽出単位（世帯・個人）と、段階的に抽出を行う。

*   **解説**:
    1.  **系統抽出の注意**
