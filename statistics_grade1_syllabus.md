# 統計検定1級 出題範囲・学習ガイド

統計検定1級は、大学専門課程（3・4年次）および大学院基礎課程で修得すべき統計学の専門理論と、実データに対する高度な応用モデリング能力を評価する、国内最高峰の統計資格試験です。

---

## 1. 試験概要

| 項目 | ① 統計数理（共通） | ② 統計応用（分野選択） |
| :--- | :--- | :--- |
| **試験方式** | 筆記試験（記述式・年1回） | 筆記試験（記述式・年1回） |
| **試験時間** | 90分 | 90分 |
| **問題構成** | 5問中 3問を選択回答 | 5問中 3問を選択回答 |
| **選択分野** | なし（数理統計学理論） | 出願時に**4つの分野**から1つを選択 |
| **合格基準** | 素点ベースで合格点以上 | 素点ベースで合格点以上 |
| **持ち込み** | 電卓（関数電卓は不可） | 電卓（関数電卓は不可） |

> [!NOTE]
> - 「統計数理」と「統計応用」の**両方に合格**することで「1級合格」となります（科目合格は10年間有効）。
> - CBT方式ではなく完全記述式であり、数式の論理的導出や証明、モデルの設計意図などの記述力が問われます。

---

## 2. 統計数理（共通必須）

統計数理では、微積分や線形代数をフルに用いた数理統計学の証明・計算問題が出題されます。

### I. 確率と確率変数
- **多次元確率変数**: 
  - 同時確率密度関数、周辺分布、条件付き分布、共分散行列の数理、変数変換（ヤコビアン計算）
- **母関数と極限定理**: 
  - 確率母関数、モーメント母関数、特性関数の性質と導出
  - 確率収束と法則収束の定義、中心極限定理の数理証明、デルタ法、スラスキーの定理
- **種々の確率分布**: 
  - 正規分布、ガンマ分布、ベータ分布、コーシー分布、ディリクレ分布、多変量正規分布の代数的性質

---

### II. 統計的推測の理論
- **点推定理論**:
  - **不偏推定量**、一致推定量、**一様最小分散不偏推定量（UMVUE）**の導出
  - **十分統計量**（ネイマン・フィッシャーの因子分解定理）、完備十分統計量、ラオ・ブラックウェルの定理
  - フィッシャー情報量（情報量行列）、**クラメール・ラオの不等式（下限）**の証明
  - 最尤推定量の漸近正規性、漸近有効性、漸近不偏性
- **仮説検定理論**:
  - 帰無仮説と対立仮説、第1種・第2種の過誤
  - **ネイマン・ピアソンの基本定理**を用いた最強力検定（UMP）の構成
  - **尤度比検定**の導出、ワルド（Wald）検定、スコア（Score）検定の数理

---

### III. 線形モデルとその他の数理
- **線形回帰の数理**: 
  - **ガウス・マルコフの定理**の証明、射影行列（ハット行列）の代数、決定係数の代数的直交分解、回帰係数の平方和の性質
- **分散分析と実験計画の理論**: 
  - 二元配置分散分析における分散成分の推論、直交配列の数学的性質
- **ノンパラメトリック検定の数理**: 
  - 符号検定、順位統計量、順位和検定の漸近正規性の導出

---

## 3. 統計応用（専門選択・4分野）

受験者は、出願時に以下の4つの分野から1つを選択し、その分野におけるデータ解析の実践力を問う記述問題を解答します。

```carousel
#### 人文科学
- **テスト理論**: 項目反応理論（IRT）、古典的テスト理論、等化、信頼性係数（クロンバックのαなど）
- **因子分析と構造方程式**: 探索的/確証的因子分析、共分散構造分析（SEM）、マルチレベルモデル（HLM）
- **多変量解析の応用**: 多次元尺度構成法（MDS）、クラスター分析、潜在クラス分析
- **カテゴリデータ解析**: 複数次元の分割表、対数線形モデル、対応分析

<!-- slide -->

#### 社会科学
- **計量経済学**: 操作変数法、パネルデータ分析（固定効果/変量効果）、同時決定バイアス、構造方程式モデル
- **時系列解析の応用**: 単位根検定（ADF検定）、共積分（コインテグレーション）、VARモデル、状態空間モデル、カルマンフィルタ、ARCH/GARCHモデル
- **経済統計**: 国民経済計算（SNA）、産業連関表、景気動向指数、人口統計（生命表、合計特殊出生率）
- **調査設計**: 複雑な標本調査のデザイン、回答バイアスの補正、欠損値の代入（多重代入法）

<!-- slide -->

#### 理工学
- **品質管理 (SQC)**: 管理図（Xbar-R, EWMA, CUSUM）、抜取検査（計量/計数）、品質工学（パラメータ設計、S/N比）
- **信頼性工学**: 故障寿命分布（ワイブル、対数正規）、ハザード関数、信頼性ブロック図、信頼度算出、コックス比例ハザードモデルの応用
- **実験計画・最適化**: 応答曲面法、配合設計、直交表の割り付け、交絡法
- **応用システム**: マルコフ決定過程、カルマンフィルタの工学的応用

<!-- slide -->

#### 医薬生物学
- **生存時間解析**: カプラン・マイヤー生存曲線、ログランク検定、コックスの比例ハザードモデル、生存時間の中央値の比較
- **臨床試験の計画**: ランダム化、二重盲検、クロスオーバー計画、非劣性試験・優越性試験、必要なサンプルサイズ計算（検出力計算）
- **疫学**: 相対リスク（RR）、オッズ比（OR）、感度・特異度、ROC曲線、交絡の調整（傾向スコア）
- **メタアナリシス**: 固定効果モデルと変量効果モデル、フォレストプロット、出版バイアス
```

## 4. 合格に向けたロードマップ

```
1. 【基礎】準1級範囲の徹底習得（『統計学実践ワークブック』の完全理解）
    └ 1級の前提として、準1級レベルのトピック（特に回帰、時系列、確率過程など）の公式や定義が頭に入っている必要があります。

2. 【数理統計】数理統計学テキストでの証明演習
    └ 『現代数理統計学』（竹村彰通 著）などの標準的なテキストを使い、十分統計量や最尤推定量、最強力検定などの数式を「何も見ずにゼロから導出できる」レベルまで繰り返し手を動かして証明します。

3. 【応用】選択分野の専門書と過去問演習
    └ 自身の専攻や実務に直結する分野を選択し、過去の記述問題を解いて「記述式での適切な答案作成の型」を身につけます。
```
