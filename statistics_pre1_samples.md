### Q1. 変数変換とヤコビアン (Ch 4)
独立な $X, Y \sim \text{Exp}(1)$ に対し、 $U = X + Y, \ V = \frac{X}{X + Y}$ としたときの同時確率密度関数 $g(u, v)$ として正しいものを選べ。

1.  $g(u, v) = u e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
2.  $g(u, v) = e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
3.  $g(u, v) = u^2 e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
4.  $g(u, v) = v e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
5.  $g(u, v) = u v e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$

*   **解答**: **1**
*   **解説**:
    $X = UV, \ Y = U(1-V)$ よりヤコビアン $|J| = U$。 $g(u, v) = e^{-x}e^{-y}|J| = e^{-u} \cdot u = u e^{-u}$。

### Q2. 積率母関数（モーメント母関数） (Ch 2)
確率変数 $X$ の確率密度関数が $f(x) = \lambda e^{-\lambda x} \ (x \geq 0, \ \lambda > 0)$ であるとき、 $X$ の積率母関数 $M_X(t) = E[e^{tX}]$ として正しいものを選べ。

1.  $M_X(t) = \frac{\lambda}{\lambda - t} \quad (t < \lambda)$
2.  $M_X(t) = \frac{1}{\lambda - t} \quad (t < \lambda)$
3.  $M_X(t) = \frac{\lambda}{\lambda + t} \quad (t > -\lambda)$
4.  $M_X(t) = e^{\lambda t}$
5.  $M_X(t) = \frac{\lambda t}{\lambda - t}$

*   **解答**: **1**
*   **解説**:
    $$M_X(t) = \int_0^\infty e^{tx} \lambda e^{-\lambda x} dx = \lambda \int_0^\infty e^{-(\lambda - t)x} dx = \lambda \left[ -\frac{e^{-(\lambda - t)x}}{\lambda - t} \right]_0^\infty = \frac{\lambda}{\lambda - t} \quad (\lambda - t > 0)$$

### Q3. 歪度と尖度の解釈 (Ch 3)
正規分布 $N(\mu, \sigma^2)$ の歪度および（正規分布を0とする）定義による超過尖度（Kurtosis Excess）の組み合わせとして正しいものを選べ。

1.  歪度: $0$、 超過尖度: $0$
2.  歪度: $0$、 超過尖度: $3$
3.  歪度: $1$、 超過尖度: $3$
4.  歪度: $1$、 超過尖度: $0$
5.  歪度: $0$、 超過尖度: $-1$

*   **解答**: **1**
*   **解説**:
    正規分布は左右対称であるため歪度（非対称性）は $0$。定義により、正規分布の尖度そのものは $3$ ですが、超過尖度（尖度 $- 3$）は $0$ となります。

### Q4. 負の二項分布の期待値 (Ch 5)
成功確率 $p \ (0 < p < 1)$ の独立な試行において、 $r$ 回成功するまでに発生する「失敗回数」 $X$ は負の二項分布に従う。この $X$ の期待値 $E[X]$ として正しいものを選べ。

1.  $E[X] = \frac{r(1-p)}{p}$
2.  $E[X] = \frac{r}{p}$
3.  $E[X] = \frac{r(1-p)}{p^2}$
4.  $E[X] = \frac{rp}{1-p}$
5.  $E[X] = \frac{r}{p^2}$

*   **解答**: **1**
*   **解説**:
    幾何分布（初めて1回成功するまでの失敗回数）の期待値は $\frac{1-p}{p}$ であり、負の二項分布はその $r$ 個の独立な和なので、期待値は $r$ 倍の $\frac{r(1-p)}{p}$ となります。

### Q5. ガンマ分布とベータ分布の性質 (Ch 6)
連続型確率分布に関する記述として、**最も不適切なもの**を選べ。

1.  形状パラメータ $\alpha = 1$ のガンマ分布は、指数分布に一致する。
2.  ベータ分布 $Beta(\alpha, \beta)$ において、 $\alpha = 1, \beta = 1$ のとき、一様分布 $U(0, 1)$ に一致する。
3.  ガンマ分布は、独立な複数の指数分布の和が従う分布である（整数の形状パラメータのとき）。
4.  ベータ分布の確率変数がとる値の範囲は、 $0 \le x \le 1$ である。
5.  ベータ分布の期待値は常に $\frac{\alpha}{\beta}$ である。

*   **解答**: **5**
*   **解説**:
    ベータ分布 $Beta(\alpha, \beta)$ の期待値は、 $\frac{\alpha}{\alpha + \beta}$ です。

### Q6. 極限定理と漸近正規性・デルタ法 (Ch 7)
確率変数 $X_n$ が $\sqrt{n}(X_n - \mu) \xrightarrow{d} N(0, \sigma^2)$ に法則収束するとき、関数 $g(x) = x^2$ に対して、デルタ法を用いて得られる $\sqrt{n}(X_n^2 - \mu^2)$ の漸近分散として正しいものを選べ。

1.  $4\mu^2 \sigma^2$
2.  $\mu^2 \sigma^2$
3.  $2\mu \sigma^2$
4.  $4\mu \sigma^2$
5.  $2\mu^2 \sigma^2$

*   **解答**: **1**
*   **解説**:
    デルタ法より、漸近分散は $[g'(\mu)]^2 \sigma^2$ になります。 $g'(x) = 2x \implies g'(\mu) = 2\mu$。したがって、漸近分散は $(2\mu)^2 \sigma^2 = 4\mu^2 \sigma^2$。

### Q8. マルコフ連鎖の定常分布 (Ch 14)
推移確率行列 $P = \begin{pmatrix} 0.8 & 0.2 \\ 0.3 & 0.7 \end{pmatrix}$ のマルコフ連鎖の定常分布ベクトル $\boldsymbol{\pi} = (\pi_1, \pi_2)$ を選べ。

1.  $(0.5, 0.5)$
2.  $(0.4, 0.6)$
3.  $(0.6, 0.4)$
4.  $(0.3, 0.7)$
5.  $(0.7, 0.3)$

*   **解答**: **3**
*   **解説**:
    $\boldsymbol{\pi} P = \boldsymbol{\pi} \implies 0.8\pi_1 + 0.3\pi_2 = \pi_1 \implies 0.3\pi_2 = 0.2\pi_1 \implies \pi_1 = 1.5\pi_2$。 $\pi_1+\pi_2=1$ より $\boldsymbol{\pi} = (0.6, 0.4)$。

### Q9. 確率過程の基礎（ポアソン過程） (Ch 15)
単位時間あたりの発生率 $\lambda$ のポアソン過程において、あるイベントが発生してから次のイベントが発生するまでの「待ち時間（間隔）」 $T$ が従う連続型確率分布として正しいものを選べ。

1.  ポアソン分布
2.  指数分布
3.  正規分布
4.  ガンマ分布
5.  ベータ分布

*   **解答**: **2**
*   **解説**:
    ポアソン過程におけるイベント発生時間の間隔 $T$ は、パラメータ $\lambda$ の**指数分布**に従います。また、 $k$ 個のイベントが発生するまでの待ち時間はガンマ分布に従います。

### Q10. 十分統計量と因子分解定理 (Ch 8)
大きさ $n$ のサンプルが区間 $[0, \theta]$ 上の一様分布に従うとき、 $\theta$ に対する十分統計量 $T$ を選べ。

1.  $T = \sum X_i$
2.  $T = \min(X_i)$
3.  $T = \max(X_i)$
4.  $T = \frac{1}{n} \sum X_i^2$
5.  $T = (\min(X_i), \max(X_i))$

*   **解答**: **3**
*   **解説**:
    尤度 $L(\theta) = \frac{1}{\theta^n} I(\max(x_i) \leq \theta) I(\min(x_i) \geq 0)$。因子分解定理より $\max(X_i)$ が十分統計量になります。

### Q11. ネイマン・ピアソンの基本定理 (Ch 10)
ある確率変数 $X$ の確率密度について、帰無仮説 $H_0: f(x) = f_0(x)$ と対立仮説 $H_1: f(x) = f_1(x)$ の単純仮説の検定を行う。
有意水準 $\alpha$ において検出力を最大にする「最強力棄却域 $R$」を構成するための条件として正しいものを選べ。
（ただし、 $k > 0$ は有意水準を $\alpha$ に合わせるための定数とする）

1.  $R = \left\{ x \ \middle| \ \frac{f_1(x)}{f_0(x)} \ge k \right\}$
2.  $R = \left\{ x \ \middle| \ \frac{f_0(x)}{f_1(x)} \ge k \right\}$
3.  $R = \left\{ x \ \middle| \ f_1(x) - f_0(x) \le k \right\}$
4.  $R = \left\{ x \ \middle| \ \frac{f_1(x)}{f_0(x)} \le k \right\}$
5.  $R = \left\{ x \ \middle| \ f_0(x) f_1(x) \ge k \right\}$

*   **解答**: **1**
*   **解説**:
    ネイマン・ピアソンの定理より、最強力検定の棄却域は、尤度比 $\frac{L(H_1)}{L(H_0)} = \frac{f_1(x)}{f_0(x)}$ が一定値以上となる領域になります。

### Q12. 尤度比適合度検定の統計量 (Ch 12)
カテゴリ数が $K$ 、各カテゴリの観測度数を $O_i$、帰無仮説の下での期待度数を $E_i$ とする。
ピアソンのカイ二乗検定統計量 $\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$ に代わって用いられる、尤度比検定統計量（G二乗） $G^2$ の定義式として正しいものを選べ。

1.  $G^2 = 2 \sum_{i=1}^K O_i \log\left(\frac{O_i}{E_i}\right)$
2.  $G^2 = 2 \sum_{i=1}^K E_i \log\left(\frac{O_i}{E_i}\right)$
3.  $G^2 = \sum_{i=1}^K O_i \log\left(\frac{O_i}{E_i}\right)$
4.  $G^2 = 2 \sum_{i=1}^K (O_i - E_i) \log\left(\frac{O_i}{E_i}\right)$
5.  $G^2 = -2 \sum_{i=1}^K O_i \log\left(\frac{E_i}{O_i}\right)$

*   **解答**: **1**
*   **解説**:
    尤度比検定統計量 $G^2$ は、完全モデルと制限モデルの対数尤度の差の $-2$ 倍より導出され、 $G^2 = 2 \sum O_i \log(O_i / E_i)$ となります。

### Q13. 標本調査法における有限母集団修正 (Ch 21)
ロット数 $N=1000$ から $n=100$ の非復元抽出を行った。母分散 $\sigma^2=900$ のとき、標本平均 $\bar{X}$ の分散 $V(\bar{X})$（有限母集団修正あり）を選べ。

1.  $9.0$
2.  $8.1$
3.  $9.9$
4.  $0.9$
5.  $81.0$

*   **解答**: **2**
*   **解説**:
    $V(\bar{X}) = \frac{\sigma^2}{n} \left( \frac{N-n}{N-1} \right) = \frac{900}{100} \left( \frac{900}{999} \right) \approx 9 \times 0.9009 \approx 8.1$。

### Q14. 不完全データ処理（EMアルゴリズム） (Ch 29)
観測 $Y$ と欠損 $Z$ に対し、ステップ $t$ で暫定値 $\theta^{(t)}$ が与えられたもとでのEステップのQ関数 $Q(\theta | \theta^{(t)})$ を選べ。

### Q15. モデル選択（AIC） (Ch 30)
モデル1（パラメータ $k=3$, $\log L = -150$）とモデル2（パラメータ $k=5$, $\log L = -147$）のAICによる比較判定として正しいものを選べ。

1.  モデル1: $306$、モデル2: $304$ で、モデル2が適している。
2.  モデル1: $306$、モデル2: $304$ で、モデル1が適している。
3.  モデル1: $-294$、モデル2: $-284$ で、モデル1が適している。
4.  モデル1: $303$、モデル2: $302$ で、モデル2が適している。
5.  モデル1: $153$、モデル2: $152$ で、モデル2が適している。

*   **解答**: **1**

    $AIC = -2\log L + 2k$。 $AIC_1 = -2(-150)+6 = 306$。 $AIC_2 = -2(-147)+10 = 304$。小さいためモデル2が適しています。

### Q22. クラスター分析（ウォード法） (Ch 24)

### Q23. 因子分析・グラフィカルモデル（d-分離） (Ch 25)

### Q29. 条件付き期待値と最小二乗予測 (Ch 3)
確率変数 $X$ と $Y$ について、 $X$ が与えられたときの $Y$ の条件付き期待値 $E[Y|X]$ は、 $X$ の任意の関数 $g(X)$ の中で平均二乗誤差 $E[(Y - g(X))^2]$ を最小にする予測量であることが知られている。
この性質を証明する数理的プロセスに関する以下の記述のうち、**最も不適切なもの**を選べ。

1.  平均二乗誤差を $E\left[ \left( (Y - E[Y|X]) + (E[Y|X] - g(X)) \right)^2 \right]$ と分解し、展開する。
2.  展開後のクロス項 $2E[(Y - E[Y|X])(E[Y|X] - g(X))]$ の期待値は、反復期待値の法則を用いて $2E_X\left[ E_{Y|X}[(Y - E[Y|X])(E[Y|X] - g(X)) | X] \right]$ と表せる。
3.  条件付き期待値の性質より、 $E_{Y|X}[(Y - E[Y|X])(E[Y|X] - g(X)) | X] = (E[Y|X] - g(X)) E_{Y|X}[Y - E[Y|X] | X]$ と変形でき、これは $0$ になる。
4.  したがって、 $E[(Y - g(X))^2] = E[(Y - E[Y|X])^2] + E[(E[Y|X] - g(X))^2]$ が成り立つ。
5.  平均二乗誤差を最小にするためには、 $g(X)$ は $Y$ の期待値 $E[Y]$ に常に一致していなければならない。

*   **解答**: **5**
*   **解説**:
    分解式 $E[(Y - g(X))^2] = E[(Y - E[Y|X])^2] + E[(E[Y|X] - g(X))^2]$ において、第2項は非負の期待値であり、これが最小（0）になるのは $g(X) = E[Y|X]$ のときです。したがって、最適な予測量は条件付き期待値 $E[Y|X]$ であり、単なる定数期待値 $E[Y]$ ではありません。よって選択肢5が不適切です。

### Q30. 対数正規分布とコーシー分布 (Ch 6)

1.  確率変数 $X$ が対数正規分布に従うとき、 $\log X$ は一様分布に従う。
2.  コーシー分布は、その期待値（平均値）および分散のいずれも数学的に定義されない（存在しない）分布として知られている。
3.  コーシー分布の確率密度関数は、標準正規分布と同様に左右対称の釣鐘型であるが、すそ（裾）の部分の減衰速度が正規分布よりも著しく速い。

### Q31. 多変量正規分布の性質 (Ch 6)
2次元確率ベクトル $\mathbf{X} = (X_1, X_2)^T$ が、平均ベクトル $\boldsymbol{\mu} = (\mu_1, \mu_2)^T$、共分散行列 $\boldsymbol{\Sigma} = \begin{pmatrix} \sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2^2 \end{pmatrix}$ の2次元正規分布に従うとする。
このとき、 $X_1 = x_1$ を与えたときの $X_2$ の条件付き期待値 $E[X_2 | X_1 = x_1]$ および条件付き分散 $V[X_2 | X_1 = x_1]$ の組み合わせとして正しいものを選べ。

1.  $E[X_2 | X_1 = x_1] = \mu_2 + \rho \frac{\sigma_2}{\sigma_1}(x_1 - \mu_1)$ , \ $V[X_2 | X_1 = x_1] = \sigma_2^2 (1 - \rho^2)$

3.  $E[X_2 | X_1 = x_1] = \mu_2 + \rho (x_1 - \mu_1)$ , \ $V[X_2 | X_1 = x_1] = \sigma_2^2 (1 - \rho^2)$
4.  $E[X_2 | X_1 = x_1] = \mu_2 + \rho \frac{\sigma_2}{\sigma_1}x_1$ , \ $V[X_2 | X_1 = x_1] = \sigma_2^2$
5.  $E[X_2 | X_1 = x_1] = \mu_2 + \frac{\sigma_2}{\sigma_1}(x_1 - \mu_1)$ , \ $V[X_2 | X_1 = x_1] = \sigma_2^2 (1 - \rho^2)$

### Q32. 枢軸量を用いた信頼区間の構成 (Ch 9)
母集団の確率密度関数が $f(x; \theta) = \frac{1}{\theta} e^{-x/\theta} \ (x \ge 0, \ \theta > 0)$ である指数分布から、サイズ $n$ の独立な標本 $X_1, \dots, X_n$ を得る。
標本平均を $\bar{X}$ とするとき、統計量 $T = \frac{2n\bar{X}}{\theta}$ は自由度 $2n$ のカイ二乗分布 $\chi^2(2n)$ に従うことが知られている。
この $T$ を枢軸量として用いたときの、パラメータ $\theta$ に対する信頼度 $95\%$ の信頼区間として正しい式を選べ。
（ただし、 $\chi^2_{0.025}(2n)$ および $\chi^2_{0.975}(2n)$ は自由度 $2n$ のカイ二乗分布の上側 $2.5\%$ 点および $97.5\%$ 点とする）

1.  $\left[ \frac{2n\bar{X}}{\chi^2_{0.025}(2n)}, \quad \frac{2n\bar{X}}{\chi^2_{0.975}(2n)} \right]$
2.  $\left[ \frac{2n\bar{X}}{\chi^2_{0.975}(2n)}, \quad \frac{2n\bar{X}}{\chi^2_{0.025}(2n)} \right]$
3.  $\left[ \frac{\bar{X}}{\chi^2_{0.025}(2n)}, \quad \frac{\bar{X}}{\chi^2_{0.975}(2n)} \right]$
4.  $\left[ \frac{2\bar{X}}{\chi^2_{0.025}(2n)}, \quad \frac{2\bar{X}}{\chi^2_{0.975}(2n)} \right]$
5.  $\left[ 2n\bar{X} \chi^2_{0.975}(2n), \quad 2n\bar{X} \chi^2_{0.025}(2n) \right]$

*   **解答**: **1**
*   **解説**:
    $T \sim \chi^2(2n)$ より、 $P\left( \chi^2_{0.975}(2n) \le \frac{2n\bar{X}}{\theta} \le \chi^2_{0.025}(2n) \right) = 0.95$ が成り立ちます。
    各辺の逆数をとると、不等号が反転します。
    $$\frac{1}{\chi^2_{0.025}(2n)} \le \frac{\theta}{2n\bar{X}} \le \frac{1}{\chi^2_{0.975}(2n)}$$
    各辺に $2n\bar{X}$ を掛けることで、 $\theta$ に対する信頼区間が得られます。
    $$\frac{2n\bar{X}}{\chi^2_{0.025}(2n)} \le \theta \le \frac{2n\bar{X}}{\chi^2_{0.975}(2n)}$$

### Q33. Wald・尤度比・Score検定の比較 (Ch 10)
パラメータ $\theta$ を持つ尤度関数 $L(\theta)$ 、対数尤度関数 $l(\theta) = \log L(\theta)$ に対し、帰無仮説 $H_0: \theta = \theta_0$ を検定する。
最尤推定量を $\hat{\theta}$ 、フィッシャー情報量を $I(\theta)$ とするとき、3つの漸近検定統計量（Wald, 尤度比 LRT, スコア Score）に関する記述として、**最も不適切なもの**を選べ。

1.  尤度比検定統計量は $LRT = -2(l(\theta_0) - l(\hat{\theta}))$ であり、帰無仮説の下で漸近的に自由度1のカイ二乗分布に従う。
2.  Wald検定統計量は $W = \frac{(\hat{\theta} - \theta_0)^2}{Var(\hat{\theta})}$ であり、最尤推定量 $\hat{\theta}$ の計算を必須とする。
3.  スコア検定統計量は $S = \frac{[l'(\theta_0)]^2}{I(\theta_0)}$ であり、最尤推定値 $\hat{\theta}$ を算出することなく実行可能である。
4.  Wald検定はパラメータの局所的な変数変換（スケール変更）に対して不変であるが、尤度比検定は変数変換によって統計量の値が変化してしまう。
5.  サンプルサイズ $n \to \infty$ の極限において、これら3つの検定統計量は確率的に同等（漸近等価）である。

*   **解答**: **4**
*   **解説**:
    実際には逆です。**尤度比検定（LRT）はパラメータの変数変換（例えば $\theta \to \eta = g(\theta)$）に対して不変**ですが、**Wald検定は変数変換に対して不変ではありません**（変換後の関数の微分を用いて分散を近似するデルタ法によるため、変換によって検定統計量の値が変わってしまいます）。したがって、選択肢4が不適切です。

### Q34. 多重比較法と有意水準の調整 (Ch 11)
3群以上の多重比較法に関する記述として、**最も不適切なもの**を選べ。

1.  $k$ 個の異なる群に対して、すべての2群間のペア（計 $m = k(k-1)/2$ 通り）について個別の有意水準 $\alpha_{ind}$ でt検定を繰り返す場合、全体の第一種の誤り（誤警報）を犯す確率（Family-wise Error Rate: FWER）は有意水準より大きくなる。
2.  ボンフェローニ（Bonferroni）の調整法では、全体の有意水準を $\alpha$ に維持するため、各検定の有意水準を $\alpha / m$ と厳しく設定する。
3.  ボンフェローニの調整は、各検定統計量の依存関係（相関）に関わらず適用できる極めて頑健な方法であるが、比較対象の数 $m$ が多くなると保守的になりすぎ（検定力が著しく低下し）差を検出しにくくなる。

*   **解答**: **5**
*   **解説**:
    ボンフェローニの補正は、ボンフェローニの不等式（ブールの不等式： $P(\bigcup A_i) \le \sum P(A_i)$ ）に基づいているため、**イベント同士の独立性や相関の有無に関わらず、どのような場合でも常に確率の上界を保証します**。独立性の仮定は不要です。よって選択肢5が不適切です。

### Q36. 共分散分析（ANCOVA）の構造と前提 (Ch 20)
共分散分析（ANCOVA）に関する記述として、**最も適切なもの**を選べ。

1.  共分散分析は、説明変数としてカテゴリー変数（因子）のみを含み、連続変数は一切含まないモデルである。
2.  共分散分析において、比較したい群間で調整変数（共変量：Covariate）の平均値に系統的な差がある場合、その共変量による調整は数学的に不可能である。
3.  共分散分析が有効に機能するための最も重要な前提条件は、因子のすべての水準にわたって「共変量と目的変数の関係（傾き）が等しい（平行である）」ことである。
4.  共変量と因子の間に交互作用が検出された場合（回帰が平行でない場合）でも、通常の共分散分析の手続きをそのまま適用して平均値の比較を行ってよい。

*   **解答**: **3**

    - 調整が正しく機能するためには、各群における共変量から目的変数への回帰の傾きが等しいという**「回帰の平行性（交互作用がないこと）」**が不可欠です（選択肢3が適切）。
    - 交互作用がある（平行でない）場合は、共分散分析ではなく、交互作用項を含めた通常の重回帰分析などとして扱う必要があります。

### Q37. 機械学習手法（SVM・ランダムフォレスト） (Ch 26)
パターン認識および機械学習の手法に関する記述として、**最も不適切なもの**を選べ。

1.  サポートベクターマシン（SVM）は、分類境界に最も近いデータ点（サポートベクター）からのマージンを最大化する超平面を構築する。
2.  SVMにおいて、元の特徴空間では線形分離不可能なデータに対し、カーネル関数を用いて暗黙的に高次元空間へ写像し分類する手法を「カーネルトリック」と呼ぶ。
3.  ランダムフォレストは、決定木を弱学習器とするバギング手法であり、過学習を抑制し安定した予測性能を持つ。
4.  ランダムフォレストを構成する個々の決定木の作成プロセスにおいて、各ノードの分割時には、常にすべての説明変数の中から最も情報利得が大きい変数が選ばれる。
5.  決定木の分割基準として用いられるジニ不純度やエントロピーは、ノード内の不確実性（クラスの混ざり具合）が低いほど小さな値をとる。

*   **解答**: **4**
*   **解説**:
    ランダムフォレストでは、個々の決定木のノードを分割する際、**すべての説明変数ではなく、ランダムに選択された説明変数のサブセット（一部の変数）**の中から最適な分割変数を探索します。これにより決定木間の相関を低下させ、モデル全体のバリアンス（ばらつき）を効果的に低減させています。よって選択肢4が不適切です。

### Q38. 生存時間解析とハザード比 (Ch 26)
生存時間解析における手法に関する記述として、**最も不適切なもの**を選べ。

1.  生存時間データにおいて、追跡不能や観察期間終了に伴いイベント発生が確認できなかったデータを「右側打ち切り」と呼ぶ。
2.  カプラン・マイヤー（Kaplan-Meier）法は、特定の生存時間分布を仮定せず、生存率を累積的に計算して階段状の生存曲線を描くノンパラメトリックな手法である。
3.  コックス（Cox）比例ハザードモデルは、ハザード比を $h(t) = h_0(t) \exp(\boldsymbol{\beta}^T \mathbf{x})$ と表すセミパラメトリックなモデルである。
4.  コックスモデルのパラメータ推定には、時間依存する基準ハザード $h_0(t)$ の形状（分布）を正規分布やワイブル分布としてあらかじめ仮定しておく必要がある。
5.  コックスモデルの比例ハザード性の仮定とは、異なる共変量を持つ2つのグループ間のハザード比が、時間 $t$ の経過に関わらず一定であるという仮定である。

*   **解答**: **4**
*   **解説**:
    コックス比例ハザードモデルは「セミパラメトリック」手法です。これは、共変量の効果部分 $\exp(\boldsymbol{\beta}^T \mathbf{x})$ にはパラメトリックな仮定を置く一方で、**基準ハザード $h_0(t)$ は特定の確率分布（ワイブル分布など）を仮定せず、完全にノンパラメトリックに（未知の任意の時間関数として）扱う**ためです。したがって、分布の事前仮定を必要とするという選択肢4が不適切です。

### Q39. 探索的因子分析と因子回転 (Ch 25)
探索的因子分析において、抽出された初期の因子負荷行列を回転させる「因子回転（Factor Rotation）」に関する記述として、**最も不適切なもの**を選べ。

1.  直交回転（Orthogonal Rotation）は、回転後の因子同士が互いに独立（無相関）であるという幾何学的制約のもとで回転を行う。
2.  斜交回転（Oblique Rotation）は、因子間の相関を許容して回転を行うため、因子同士に関連があるような心理データなどの分析で頻繁に用いられる。
3.  バリマックス（Varimax）回転は、代表的な直交回転であり、各因子の因子負荷量の分散を最大化するように回転を行うことで、解釈の単純化を図る。
4.  プロマックス（Promax）回転は、代表的な斜交回転であり、最初にバリマックス回転を行った負荷行列を何乗かすることで対比を強調し、さらに因子間の相関を考慮して回転する。
5.  斜交回転を施すことで、各変数の共通性（因子の説明割合）およびモデルの適合度統計量は、直交回転のときから大幅に改善（変化）する。

*   **解答**: **5**
*   **解説**:
    因子回転（直交回転・斜交回転を問わず）は、因子負荷行列の座標軸を変換する処理ですが、**各変数の共通性（Communality：因子によって説明される分散）や独自性（Uniqueness）、およびモデルの残差共分散構造（適合度統計量）は、回転の前後で数学的に全く変化せず、完全に一定に保たれます**。変化するのは解釈性のみです。よって選択肢5が不適切です。

### Q40. 時系列モデル（MA・ARMA） (Ch 27)
1次移動平均モデル MA(1): $Y_t = \mu + \epsilon_t - \theta \epsilon_{t-1}$ および 自己回帰移動平均モデル ARMA(1, 1): $Y_t = \phi Y_{t-1} + \epsilon_t - \theta \epsilon_{t-1}$ （ただし $\epsilon_t \sim W.N.(0, \sigma^2)$）を考える。
これらのモデルの自己相関関数（ACF）の性質に関する記述として、**最も適切なもの**を選べ。

1.  MA(1)モデルの自己相関関数は、すべてのラグ $k \ge 1$ でなだらかに減衰する。
2.  ARMA(1, 1)モデルの自己相関関数は、ラグ $k = 1$ でのみ非ゼロであり、ラグ $k \ge 2$ では完全に $0$ になる。
3.  MA(1)モデルの自己相関関数は、ラグ $k = 1$ でのみ非ゼロであり、ラグ $k \ge 2$ では完全に $0$ になる。
4.  ARMA(1, 1)が定常であるための条件は、自己回帰パラメータ $\phi$ と移動平均パラメータ $\theta$ の双方が、 $|\phi| < 1$ かつ $|\theta| < 1$ を満たすことである。
5.  MA(1)モデルが定常であるためには、パラメータ $\theta$ の絶対値が 1 未満である必要がある。

*   **解答**: **3**
*   **解説**:
    - MA(q)モデルの自己相関関数は、ラグ $k > q$ で完全に0になる（カットオフ）という特性を持ちます。したがってMA(1)はラグ1でのみ非ゼロ、ラグ2以降は0となります（選択肢3が正しい）。
    - ARMA(1, 1)の自己相関関数は、AR成分の働きにより、ラグ2以降も指数関数的になだらかに減衰します。
    - MA(1)モデルはパラメータ $\theta$ の値に関わらず**常に定常**です（定常性条件は不要ですが、一意にパラメータを特定するための「反転可能性条件」として $|\theta| < 1$ が課されます）。
    - ARMA(1, 1)の定常条件はAR部分にのみ依存するため、 $|\phi| < 1$ です（ $\theta$ は影響しません）。

### Q41. 周波数領域の分析（スペクトル解析） (Ch 27)
定常時系列のスペクトル解析に関する記述として、**最も不適切なもの**を選べ。

1.  定常時系列の自己共分散関数 $\gamma_k$ をフーリエ変換（または余弦変換）することにより、周波数ごとの分散の寄与を示す「スペクトル密度関数（パワー・スペクトル）」が得られる。
2.  スペクトル密度関数 $f(\lambda)$ を全周波数領域 $[-\pi, \pi]$ で積分した値は、時系列全体の分散 $\gamma_0 = V[Y_t]$ に一致する。
3.  ホワイトノイズのスペクトル密度関数は、すべての周波数において一定値（平坦）をとる。
4.  有限の観測データ $\{y_1, \dots, y_n\}$ から、各周波数での周期成分の強さをサンプルデータとして計算する関数を「ピリオドグラム」と呼ぶ。
5.  12ヶ月の周期的な季節変動が非常に顕著な月次データをスペクトル解析すると、周波数 $\lambda = \frac{\pi}{6}$ においてスペクトル密度関数は極小値をとる。

*   **解答**: **5**
*   **解説**:
    時系列に周期 $T$ の規則的な変動が存在する場合、対応する周波数 $\lambda = \frac{2\pi}{T}$ において、スペクトル密度は非常に大きな値（**極大値・ピーク**）を示します。
    周期 $T=12$（ヶ月）のとき、 $\lambda = \frac{2\pi}{12} = \frac{\pi}{6}$ となるため、この周波数においてスペクトル密度関数は鋭い極大値をとります。極小値をとるという選択肢5が誤りです。

### Q42. MCMC法のサンプリングアルゴリズム (Ch 31)
ベイズ統計で用いられるマルコフ連鎖モンテカルロ（MCMC）法に関する記述として、**最も不適切なもの**を選べ。

1.  MCMC法は、目標とする事後分布を定常分布として持つマルコフ連鎖をシミュレーションし、その収束後の連鎖からサンプルを得る方法である。
2.  ギブスサンプリング（Gibbs Sampling）は、多変量の結合事後分布からサンプリングを行う際、各パラメータを「他のすべてのパラメータを条件付けたフル条件付き事後分布」から1つずつ交互に生成する手法である。
3.  メトロポリス・ヘイスティングス（MH）法では、提案分布から候補値を発生させ、事後確率密度の比に基づく受容確率 $\alpha$ によって確率的に採択または棄却を行う。
4.  MCMCの初期段階は連鎖が定常分布に達しておらず初期値に依存するため、この不安定な期間（バーンイン期間）のサンプルは解析から除外する。
5.  ギブスサンプリングは、結合事後分布に含まれる各パラメータのフル条件付き分布が標準的でない（直接サンプリングできない複雑な関数である）場合でも、常に受容計算を挟まないため最も効率が良い。

*   **解答**: **5**
*   **解説**:
    ギブスサンプリングは、すべてのパラメータのフル条件付き分布が既知の標準的な分布（正規分布やベータ分布など）になり、そこから容易に直接乱数をサンプリングできる場合に限り非常に効率的です。
    しかし、**フル条件付き分布が標準的でない複雑な分布の場合、ギブスサンプリング単体では直接サンプリングが行えません**。その場合は、個々のパラメータ更新にMHサンプラーを内包する（Metropolis-within-Gibbs）などのハイブリッドなアプローチが必要になり、採択・棄却の計算が生じます。したがって、常に受容計算が不要で最も効率的であるとする選択肢5が不適切です。
