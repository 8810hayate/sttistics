## 1. 確率と確率分布分野
### 📌 キーワード：変数変換とヤコビアン (Ch 4)

#### 【問題】
互いに独立にパラメータ（期待値）が $1$ の指数分布に従う2つの確率変数 $X, Y$ がある。
それぞれの確率密度関数は $f_X(x) = e^{-x} \ (x \geq 0)$、 $f_Y(y) = e^{-y} \ (y \geq 0)$ である。
ここで、新たな確率変数 $U, V$ を以下のように定義する。
$$U = X + Y$$
$$V = \frac{X}{X + Y}$$
このとき、$U$ と $V$ の同時確率密度関数 $g(u, v)$ として正しいものを以下の選択肢から選べ。

1.  $g(u, v) = u e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
2.  $g(u, v) = e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
3.  $g(u, v) = u^2 e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
4.  $g(u, v) = v e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$
5.  $g(u, v) = u v e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$

#### 【解答】
**1. $g(u, v) = u e^{-u} \quad (u \geq 0, \ 0 \leq v \leq 1)$**

#### 【解説】
1.  **変数 $X, Y$ を $U, V$ で表す**
    与えられた定義式より、
    $$U = X + Y$$
    $$V = \frac{X}{U} \implies X = U V$$
    これらを $Y = U - X$ に代入すると、
    $$Y = U - U V = U(1 - V)$$
    変数の範囲は、$x \geq 0, y \geq 0$ より $u \geq 0$ かつ $0 \leq v \leq 1$ です。

2.  **ヤコビ行列式（ヤコビアン）を計算する**
    偏微分を行います。
    $$\frac{\partial X}{\partial U} = V, \quad \frac{\partial X}{\partial V} = U$$
    $$\frac{\partial Y}{\partial U} = 1 - V, \quad \frac{\partial Y}{\partial V} = -U$$
    ヤコビアン $J$ は以下のように計算されます。
    $$J = \det \begin{pmatrix} \frac{\partial X}{\partial U} & \frac{\partial X}{\partial V} \\ \frac{\partial Y}{\partial U} & \frac{\partial Y}{\partial V} \end{pmatrix} = \det \begin{pmatrix} V & U \\ 1-V & -U \end{pmatrix} = V(-U) - U(1-V) = -UV - U + UV = -U$$
    絶対値をとると $|J| = U$ となります（$u \geq 0$ より）。

3.  **同時確率密度関数 $g(u, v)$ を求める**
    元の独立な $X, Y$ の同時確率密度関数は $f(x, y) = e^{-x} e^{-y} = e^{-(x+y)}$ です。
    これに変数変換を適用します。
    $$g(u, v) = f(x(u,v), y(u,v)) \cdot |J|$$
    $$g(u, v) = e^{-(uv + u(1-v))} \cdot U = e^{-u} \cdot U = u e^{-u}$$
    
    よって、同時確率密度関数は $g(u, v) = u e^{-u}$ となり、選択肢 **1** が正解です。
    *(※これは、$U \sim \text{Gamma}(2, 1)$ と $V \sim \text{Beta}(1, 1)$ が互いに独立であることを意味しています。)*

## 2. 統計的推測分野
### 📌 キーワード：十分統計量と因子分解定理 (Ch 8)

#### 【問題】
大きさが $n$ のランダムサンプル $X_1, X_2, \dots, X_n$ が、区間 $[0, \theta]$ 上の一様分布に従っている。
ここで $\theta > 0$ は未知のパラメータである。各 $X_i$ の確率密度関数は以下のように表される。
$$f(x_i; \theta) = \frac{1}{\theta} I(0 \leq x_i \leq \theta)$$
（ただし、$I(\cdot)$ は指示関数で、条件を満たすとき $1$、満たさないとき $0$ をとる）
このとき、パラメータ $\theta$ に対する十分統計量 $T$ として正しいものを以下の選択肢から選べ。

1.  $T = \sum_{i=1}^n X_i$
2.  $T = \min(X_1, \dots, X_n)$
3.  $T = \max(X_1, \dots, X_n)$
4.  $T = \frac{1}{n} \sum_{i=1}^n X_i^2$
5.  $T = (\min(X_i), \max(X_i))$ のペア

#### 【解答】
**3. $T = \max(X_1, \dots, X_n)$**

#### 【解説】
1.  **同時確率密度関数（尤度関数）を記述する**
    サンプルが独立であるため、同時確率密度関数は各確率密度関数の積になります。
    $$f(x_1, \dots, x_n; \theta) = \prod_{i=1}^n \left( \frac{1}{\theta} I(0 \leq x_i \leq \theta) \right) = \frac{1}{\theta^n} \prod_{i=1}^n I(0 \leq x_i \leq \theta)$$

2.  **指示関数の積を整理する**
    すべての $i$ について $0 \leq x_i \leq \theta$ が成り立つということは、最小値が $0$ 以上であり、かつ最大値が $\theta$ 以下であることと同値です。
    すなわち、
    $$\prod_{i=1}^n I(0 \leq x_i \leq \theta) = I(\min(x_i) \geq 0) \cdot I(\max(x_i) \leq \theta)$$
    よって、同時確率は以下のように書けます。
    $$f(x_1, \dots, x_n; \theta) = \frac{1}{\theta^n} I(\max(x_i) \leq \theta) \cdot I(\min(x_i) \geq 0)$$

3.  **フィッシャー・ネイマンの因子分解定理を適用する**
    因子分解定理より、同時確率密度関数が以下の形に分解できれば、$T(x_1, \dots, x_n)$ は $\theta$ に対する十分統計量です。
    $$f(x_1, \dots, x_n; \theta) = g(T(x_1, \dots, x_n), \theta) \cdot h(x_1, \dots, x_n)$$
    ここで、
    - $g(T, \theta) = \frac{1}{\theta^n} I(\max(x_i) \leq \theta)$
    - $h(x_1, \dots, x_n) = I(\min(x_i) \geq 0)$
    とおくと、関数 $g$ は統計量 $T = \max(X_1, \dots, X_n)$ と $\theta$ のみに依存し、関数 $h$ は $\theta$ に依存しません。

    したがって、十分統計量は最大値である $T = \max(X_1, \dots, X_n)$ となり、選択肢 **3** が正解です。

## 3. 回帰分析と分散分析分野
### 📌 キーワード：回帰診断法・多重共線性 (VIF) (Ch 17)

#### 【問題】
説明変数が2つ（$X_1$ と $X_2$）である重回帰モデルを推定する。
$X_1$ を被説明変数とし、$X_2$ を説明変数とする単回帰分析を行ったところ、決定係数（決定指数） $R_1^2$ は $0.90$ であった。
このとき、この重回帰モデルにおける説明変数 $X_1$ に対応する分散膨張係数（VIF: Variance Inflation Factor）として正しいものを以下の選択肢から選べ。

1.  $1.11$
2.  $2.00$
3.  $5.00$
4.  $10.00$
5.  $20.00$

#### 【解答】
**4. $10.00$**

#### 【解説】
1.  **VIFの定義**
    重回帰分析におけるある説明変数 $X_j$ のVIFは、他のすべての説明変数で $X_j$ を回帰したときの決定係数を $R_j^2$ とすると、以下のように定義されます。
    $$\text{VIF}_j = \frac{1}{1 - R_j^2}$$

2.  **VIFの計算**
    本問では説明変数が $X_1$ と $X_2$ の2つだけなので、他の説明変数で回帰した決定係数 $R_1^2$ は、まさに $X_1$ を $X_2$ で回帰した単回帰の決定係数 $0.90$ になります。
    $$\text{VIF}_1 = \frac{1}{1 - R_1^2} = \frac{1}{1 - 0.90} = \frac{1}{0.10} = 10.00$$

    VIFが $10$ 以上である場合、**強い多重共線性**が存在している可能性が疑われます。よって、正解は **4. $10.00$** です。

## 4. 質的回帰・線形モデル分野
### 📌 キーワード：ロジスティック回帰（オッズ比） (Ch 18)

#### 【問題】
ある病気（発症 $= 1$, 未発症 $= 0$）の発生確率 $p$ と、特定の生活習慣（習慣あり $= 1$, 習慣なし $= 0$）の有無を表すダミー変数 $X$ の関係を調査するため、以下のロジスティック回帰モデルを推定した。
$$\log\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 X$$
推定の結果、回帰係数の値は $\beta_0 = -3.00, \ \beta_1 = 0.693$ であった。
この生活習慣が「ある」グループの発症オッズは、「ない」グループの発症オッズの何倍になっているか、最も近いものを以下の選択肢から選べ。
（ただし、 $\log$ は自然対数であり、 $e^{0.693} \approx 2.00$ とする）

1.  $0.50$倍
2.  $1.50$倍
3.  $2.00$倍
4.  $2.71$倍
5.  $4.00$倍

#### 【解答】
**3. $2.00$倍**

#### 【解説】
1.  **オッズの表現**
    ロジスティック回帰モデルの左辺 $\log\left(\frac{p}{1-p}\right)$ は「対数オッズ（ロジット）」です。
    両辺の指数（$e$ の肩に乗せる）をとると、オッズ自体が得られます。
    $$\text{Odds} = \frac{p}{1-p} = e^{\beta_0 + \beta_1 X} = e^{\beta_0} \cdot (e^{\beta_1})^X$$

2.  **オッズ比（OR）の計算**
    オッズ比は次のようになります。
    $$\text{OR} = \frac{\text{Odds}_{X=1}}{\text{Odds}_{X=0}} = \frac{e^{\beta_0 + \beta_1}}{e^{\beta_0}} = e^{\beta_1}$$
    与えられた係数 $\beta_1 = 0.693$ を代入すると、
    $$\text{OR} = e^{0.693} \approx 2.00$$

    したがって、発症オッズは $2.00$ 倍となり、選択肢 **3** が正解です。

## 5. 多変量解析分野（次元削減）
### 📌 キーワード：主成分分析（寄与率） (Ch 22)

#### 【問題】
ある5種類のデータからなる多変数データに対して主成分分析（PCA）を行ったところ、共分散行列から得られた固有値は以下のようになった。
- 第1主成分の固有値: $\lambda_1 = 2.40$
- 第2主成分の固有値: $\lambda_2 = 1.10$
- 第3主成分の固有値: $\lambda_3 = 0.80$
- 第4主成分の固有値: $\lambda_4 = 0.50$
- 第5主成分の固有値: $\lambda_5 = 0.20$

第2主成分までの累積寄与率として最も近いものを以下の選択肢から選べ。

1.  $48\%$
2.  $64\%$
3.  $70\%$
4.  $83\%$
5.  $90\%$

#### 【解答】
**3. $70\%$**

#### 【解説】
1.  **総分散（全固有値の和）を計算する**
    $$\text{総分散} = \lambda_1 + \lambda_2 + \lambda_3 + \lambda_4 + \lambda_5 = 2.40 + 1.10 + 0.80 + 0.50 + 0.20 = 5.00$$

2.  **第2主成分までの累積寄与率を計算する**
    第2主成分までの累積寄与率は、第1主成分と第2主成分の固有値の和が総分散に占める割合です。
    $$\text{累積寄与率} = \frac{\lambda_1 + \lambda_2}{\text{総分散}} = \frac{2.40 + 1.10}{5.00} = \frac{3.50}{5.00} = 0.70 \quad (70\%)$$

    よって、累積寄与率は $70\%$ となり、選択肢 **3** が正解です。

## 6. 多変量解析分野（分類）
### 📌 キーワード：判別分析（線形判別関数） (Ch 23)

#### 【問題】
2つのグループ（グループ1、グループ2）を分類するため、2つの変数 $X_1, X_2$ を用いて線形判別分析を行った。
両グループの共分散行列は等しいと仮定し、推定されたフィッシャーの線形判別関数 $W(X_1, X_2)$ は以下のようになった。
$$W(X_1, X_2) = 2.0 X_1 + 3.0 X_2 - 12.0$$
ここで、 $W > 0$ であればグループ1、 $W < 0$ であればグループ2に分類する。
ある新規サンプル A のデータが $(X_1, X_2) = (3.0, 3.0)$ であるとき、このサンプルの判別得点（判別関数の値）および分類先として正しいものを以下の選択肢から選べ。

1.  判別得点: $-3.0$、 分類先: グループ2
2.  判別得点: $+3.0$、 分類先: グループ1
3.  判別得点: $+3.0$、 分類先: グループ2
4.  判別得点: $+1.5$、 分類先: グループ1
5.  判別得点: $-1.5$、 分類先: グループ2

#### 【解答】
**2. 判別得点: $+3.0$、 分類先: グループ1**

#### 【解説】
1.  **判別得点の計算**
    新規サンプル A のデータ $(X_1, X_2) = (3.0, 3.0)$ を線形判別関数 $W(X_1, X_2)$ に代入します。
    $$W(3.0, 3.0) = 2.0 \times 3.0 + 3.0 \times 3.0 - 12.0 = 3.0$$

2.  **分類先の判定**
    計算された判別得点は $W = 3.0 > 0$ であるため、判別基準に基づき**グループ1**に分類されます。

    したがって、正解は **2** となります。

## 7. 時系列解析・確率過程分野
### 📌 キーワード：AR(1)モデルと自己相関関数 (Ch 27)

#### 【問題】
平均が $0$、自己共分散関数が定常である次の一階自己回帰モデル AR(1) を考える。
$$Y_t = \phi Y_{t-1} + \epsilon_t \quad (t = 1, 2, \dots)$$
ここで $\epsilon_t$ は平均 $0$、分散 $\sigma^2$ の互いに独立なホワイトノイズであり、 $Y_{t-1}$ とは独立である。
この時系列が**弱定常**となる条件、および自己相関関数（ラグ $k$ ） $\rho_k = \text{Corr}(Y_t, Y_{t-k})$ の正しい組み合わせを以下の選択肢から選べ。

1.  定常条件: $\phi > 0$　/　自己相関: $\rho_k = \phi k$
2.  定常条件: $|\phi| < 1$　/　自己相関: $\rho_k = \phi^k$
3.  定常条件: $|\phi| < 1$　/　自己相関: $\rho_k = k^\phi$
4.  定常条件: $\phi < 1$　/　自己相関: $\rho_k = \phi^k$
5.  定常条件: $|\phi| \leq 1$　/　自己相関: $\rho_k = \phi^k$

#### 【解答】
**2. 定常条件: $|\phi| < 1$　/　自己相関: $\rho_k = \phi^k$**

#### 【解説】
1.  **定常条件の導出**
    特性方程式は $1 - \phi z = 0 \implies z = 1/\phi$ です。根の絶対値が $1$ より大きいことが定常条件（単位円の外側）であるため、
    $$\left| \frac{1}{\phi} \right| > 1 \implies |\phi| < 1$$

2.  **自己相関関数の導出**
    定常時系列の仮定のもとで自己共分散 $R_k = E[Y_t Y_{t-k}]$ を計算すると $R_k = \phi R_{k-1}$ となります。
    両辺を分散 $R_0$ で割ることで、自己相関関数の関係式 $\rho_k = \phi \rho_{k-1}$ が得られ、境界条件 $\rho_0 = 1$ より、 $\rho_k = \phi^k \quad (k \geq 0)$ です。

    よって、正しい組み合わせは **2** となります。

## 8. 不完全データ処理分野
### 📌 キーワード：EMアルゴリズム（Q関数） (Ch 29)

#### 【問題】
観測データ $Y$ と欠損データ $Z$ があり、完全データ $(Y, Z)$ の同時確率密度関数を $f(y, z; \theta)$ とする（$\theta$ は未知パラメータ）。
EMアルゴリズムにおいて、ステップ $t$ でのパラメータ更新値を $\theta^{(t)}$ とするとき、Eステップ（期待値ステップ）で最大化対象として計算される **Q関数** $Q(\theta | \theta^{(t)})$ の定義式として正しいものを以下の選択肢から選べ。
（ただし、 $E_{\theta}[\cdot]$ はパラメータが $\theta$ であるときの期待値を示し、対数尤度を最大化するものとする）

1.  $Q(\theta | \theta^{(t)}) = E_{\theta^{(t)}} [ \log f(Y, Z; \theta) | Y ]$
2.  $Q(\theta | \theta^{(t)}) = E_{\theta} [ \log f(Y, Z; \theta^{(t)}) | Y ]$
3.  $Q(\theta | \theta^{(t)}) = \log \int f(Y, z; \theta) dz$
4.  $Q(\theta | \theta^{(t)}) = E_{\theta^{(t)}} [ f(Y, Z; \theta) | Y ]$
5.  $Q(\theta | \theta^{(t)}) = E_{\theta} [ \log f(Y, Z; \theta) | Z ]$

#### 【解答】
**1. $Q(\theta | \theta^{(t)}) = E_{\theta^{(t)}} [ \log f(Y, Z; \theta) | Y ]$**

#### 【解説】
1.  **EMアルゴリズムの原理**
    E（Expectation）ステップの目的は、現在のパラメータ暫定値 $\theta^{(t)}$ と観測データ $Y = y$ が与えられたもとでの、完全データ対数尤度 $\log f(y, Z; \theta)$ の条件付き期待値を計算することです。
    $$Q(\theta | \theta^{(t)}) = E_{Z | Y; \theta^{(t)}} [ \log f(Y, Z; \theta) | Y = y ]$$
    したがって、正しい定義式は **1** となります。

## 9. ベイズ法分野
### 📌 キーワード：共役事前分布（ベータ・二項モデル） (Ch 31)

#### 【問題】
あるコインを投げたときに表が出る確率を $\theta$ とし、ベイズ推定を行う。
事前分布として、以下のベータ分布 $Beta(\alpha, \beta)$ を仮定する。
$$\pi(\theta) \propto \theta^{\alpha - 1} (1-\theta)^{\beta - 1} \quad (0 \leq \theta \leq 1)$$
コインを $n$ 回投げたところ、表が $x$ 回出現した。
このときの $\theta$ に対する事後分布 $\pi(\theta | x)$ として正しいベータ分布を以下の選択肢から選べ。

1.  $Beta(\alpha + x, \beta + n)$
2.  $Beta(\alpha + x, \beta + n - x)$
3.  $Beta(\alpha + n, \beta + x)$
4.  $Beta(\alpha + n - x, \beta + x)$
5.  $Beta(\alpha \cdot x, \beta \cdot (n-x))$

#### 【解答】
**2. $Beta(\alpha + x, \beta + n - x)$**

#### 【解説】
1.  **尤度と事前分布の積**
    $$\pi(\theta | x) \propto L(x | \theta) \pi(\theta) \propto \theta^x (1-\theta)^{n-x} \times \theta^{\alpha - 1} (1-\theta)^{\beta - 1} = \theta^{\alpha + x - 1} (1-\theta)^{\beta + n - x - 1}$$
    これは、パラメータが $\alpha' = \alpha + x$、 $\beta' = \beta + n - x$ のベータ分布 $Beta(\alpha + x, \beta + n - x)$ の密度関数のカーネルに一致します。

    よって、正解は **2** です。

## 10. ノンパラメトリック法分野
### 📌 キーワード：ウィルコクソンの順位和検定 (Ch 13)

#### 【問題】
2つの独立したグループA（サンプルサイズ $n_1 = 4$）とグループB（サンプルサイズ $n_2 = 4$）のデータがある。
値を小さい順に並べたところ、各サンプルの属するグループと順位（タイなし）は以下のようになった。
- 順位 1: グループA
- 順位 2: グループA
- 順位 3: グループB
- 順位 4: グループA
- 順位 5: グループB
- 順位 6: グループA
- 順位 7: グループB
- 順位 8: グループB

グループAの順位和を $W_A$ とするとき、 $W_A$ の値として正しいものを以下の選択肢から選べ。

1.  $10$
2.  $13$
3.  $16$
4.  $23$
5.  $26$

#### 【解答】
**2. $13$**

#### 【解説】
1.  **順位和の計算**
    ウィルコクソンの順位和検定では、それぞれのグループの順位を単純に足し合わせます。
    グループAに属するデータの順位は、 **1, 2, 4, 6** です。
    よって、グループAの順位和 $W_A$ は：
    $$W_A = 1 + 2 + 4 + 6 = 13$$

    （なお、グループBの順位和 $W_B$ は $3 + 5 + 7 + 8 = 23$ となります。両者の合計は $13 + 23 = 36$ であり、全体の順位和の公式 $n(n+1)/2 = 8 \times 9 / 2 = 36$ に一致します）

    したがって、正解は **2. $13$** です。

## 11. 確率過程分野
### 📌 キーワード：マルコフ連鎖の定常分布 (Ch 14)

#### 【問題】
あるシステムの2つの状態（状態1、状態2）の間の推移確率行列 $P$ が以下のように与えられている。
$$P = \begin{pmatrix} 0.8 & 0.2 \\ 0.3 & 0.7 \end{pmatrix}$$
（ここで、$P_{ij}$ は状態 $i$ から状態 $j$ へ推移する確率を示す）
このマルコフ連鎖の定常分布（極限分布）確率ベクトル $\boldsymbol{\pi} = (\pi_1, \pi_2)$ として正しいものを以下の選択肢から選べ。

1.  $\boldsymbol{\pi} = (0.5, 0.5)$
2.  $\boldsymbol{\pi} = (0.4, 0.6)$
3.  $\boldsymbol{\pi} = (0.6, 0.4)$
4.  $\boldsymbol{\pi} = (0.3, 0.7)$
5.  $\boldsymbol{\pi} = (0.7, 0.3)$

#### 【解答】
**3. $\boldsymbol{\pi} = (0.6, 0.4)$**

#### 【解説】
1.  **定常方程式の構築**
    定常分布の条件 $\boldsymbol{\pi} P = \boldsymbol{\pi}$ および確率の総和条件 $\pi_1 + \pi_2 = 1$ を解きます。
    $$(\pi_1, \pi_2) \begin{pmatrix} 0.8 & 0.2 \\ 0.3 & 0.7 \end{pmatrix} = (\pi_1, \pi_2)$$
    これより、
    $$0.8 \pi_1 + 0.3 \pi_2 = \pi_1 \implies 0.3 \pi_2 = 0.2 \pi_1 \implies 3 \pi_2 = 2 \pi_1 \implies \pi_1 = 1.5 \pi_2$$

2.  **総和条件の適用**
    $$\pi_1 + \pi_2 = 1 \implies 1.5 \pi_2 + \pi_2 = 1 \implies 2.5 \pi_2 = 1 \implies \pi_2 = 0.4$$
    $$\pi_1 = 1.5 \times 0.4 = 0.6$$

    したがって、定常分布は $\boldsymbol{\pi} = (0.6, 0.4)$ となり、選択肢 **3** が正解です。

## 12. 標本調査法分野
### 📌 キーワード：有限母集団修正 (Ch 21)

#### 【問題】
製品総数が $N = 1000$ 個ある大きなロットから、非復元抽出法を用いて大きさ $n = 100$ のランダムサンプル $X_1, \dots, X_{100}$ を抽出した。
製品全体の強度の母分散が $\sigma^2 = 900$ であるとき、標本平均 $\bar{X} = \frac{1}{n}\sum X_i$ の分散 $V(\bar{X})$ の値（有限母集団修正を考慮する）として正しいものを以下の選択肢から選べ。

1.  $9.0$
2.  $8.1$
3.  $9.9$
4.  $0.9$
5.  $81.0$

#### 【解答】
**2.  $8.1$**

#### 【解説】
1.  **有限母集団修正 (FPC) の公式**
    母集団サイズ $N$ からサイズ $n$ の非復元抽出を行う場合、標本平均の分散 $V(\bar{X})$ は以下の有限母集団修正因子を掛け合わせて計算します。
    $$V(\bar{X}) = \frac{\sigma^2}{n} \left( \frac{N - n}{N - 1} \right)$$
    （※ $N$ が極めて大きい復元抽出や無限母集団の場合は $\frac{\sigma^2}{n}$ となります）

2.  **分散の計算**
    値を代入します。ここでは $N=1000$, $n=100$, $\sigma^2 = 900$ です。
    $$V(\bar{X}) = \frac{900}{100} \left( \frac{1000 - 100}{1000 - 1} \right) = 9 \times \frac{900}{999} \approx 9 \times 0.9009 \approx 8.108$$

    最も近い値は **2. $8.1$** です。

## 13. 因子分析・グラフィカルモデル分野
### 📌 キーワード：条件付き独立性と有向グラフ（d-分離） (Ch 25)

#### 【問題】
3つの確率変数 $X, Y, Z$ の関係が、以下の有向非巡回グラフ（DAG）で表されるベイジアンネットワークのモデル（構造方程式）に従っているとする。
$$X \rightarrow Y \rightarrow Z$$
このとき、変数間の独立性または条件付き独立性に関する記述として正しいものを以下の選択肢から選べ。

1.  $X$ と $Z$ は無条件で独立である。
2.  $Y$ が与えられた（観測された）とき、 $X$ と $Z$ は条件付き独立である。
3.  $X$ が与えられたとき、 $Y$ と $Z$ は条件付き独立である。
4.  $Z$ が与えられたとき、 $X$ と $Y$ は条件付き独立である。
5.  どのような条件のもとでも、 $X, Y, Z$ はすべて互いに独立である。

#### 【解答】
**2.  $Y$ が与えられた（観測された）とき、 $X$ と $Z$ は条件付き独立である。**

#### 【解説】
1.  **グラフィカルモデルにおけるパスのブロック**
    グラフ構造 $X \rightarrow Y \rightarrow Z$ は「一方向の連鎖（chain）」構造です。
    中間変数 $Y$ が与えられていないとき、 $X$ と $Z$ は無条件で独立ではありません（$X$ の情報が $Y$ を経由して $Z$ に流れるため）。
    
2.  **d-分離（d-separation）ルール**
    中間にある変数 $Y$ を条件づける（観測値として固定する）と、 $X$ から $Z$ へのパスがブロック（遮断）され、条件付き独立 $X \perp\!\!\perp Z \mid Y$ が成立します。これはマルコフ性（現在の状態が与えられたとき、過去と未来は独立）に対応します。

    したがって、正しい記述は **2** となります。

## 14. その他の多変量解析・機械学習分野
### 📌 キーワード：決定木（ジニ不純度） (Ch 26)

#### 【問題】
決定木の分類木モデルにおいて、あるノードにあるデータ（総数 $N = 20$ 個）が、グループAに $15$ 個、グループBに $5$ 個分類されている。
このノードにおける**ジニ不純度（Gini Impurity）**の値として正しいものを以下の選択肢から選べ。

1.  $0.250$
2.  $0.375$
3.  $0.500$
4.  $0.625$
5.  $0.750$

#### 【解答】
**2.  $0.375$**

#### 【解説】
1.  **ジニ不純度の定義式**
    クラス（カテゴリ）の数を $K$、ノード内の各クラスの出現割合を $p_i \ (i=1, \dots, K)$ とすると、ジニ不純度 $I_G$ は以下のように定義されます。
    $$I_G = 1 - \sum_{i=1}^K p_i^2$$

2.  **ジニ不純度の計算**
    ここでは $K = 2$、各クラスの割合は：
    - クラスA: $p_A = \frac{15}{20} = 0.75$
    - クラスB: $p_B = \frac{5}{20} = 0.25$
    
    これらを定義式に代入します。
    $$I_G = 1 - (p_A^2 + p_B^2) = 1 - (0.75^2 + 0.25^2) = 1 - (0.5625 + 0.0625) = 1 - 0.625 = 0.375$$

    よって、ジニ不純度は **2. $0.375$** となります。（※もし完全に一方のクラスだけで占められているノードであれば、不純度は最小値 $0$ になります）

## 15. モデル選択分野
### 📌 キーワード：赤池情報量基準（AIC） (Ch 30)

#### 【問題】
あるデータ（サンプルサイズ $n = 100$）に対して、2つの異なる重回帰モデル（モデル1、モデル2）を推定したところ、以下の結果が得られた。
- **モデル1**: 説明変数 2個（パラメータ数 $k=3$（切片含む））、最大対数尤度 $\log L = -150$
- **モデル2**: 説明変数 4個（パラメータ数 $k=5$（切片含む））、最大対数尤度 $\log L = -147$

それぞれのモデルに対する赤池情報量基準（AIC）を計算し、モデル選択を行った結果として正しいものを以下の選択肢から選べ。

1.  モデル1のAICは $306$、モデル2のAICは $304$ であり、モデル2のほうが適している。
2.  モデル1のAICは $306$、モデル2のAICは $304$ であり、モデル1のほうが適している。
3.  モデル1のAICは $-294$、モデル2のAICは $-284$ であり、モデル1のほうが適している。
4.  モデル1のAICは $303$、モデル2のAICは $302$ であり、モデル2のほうが適している。
5.  モデル1のAICは $153$、モデル2のAICは $152$ であり、モデル2のほうが適している。

#### 【解答】
**1.  モデル1のAICは $306$、モデル2のAICは $304$ であり、モデル2のほうが適している。**

#### 【解説】
1.  **AICの定義式**
    赤池情報量基準（AIC）は、最大対数尤度を $\log L$、推定されたパラメータ数を $k$ とすると次のように定義されます。
    $$\text{AIC} = -2 \log L + 2k$$
    AICは値が**小さいほど**モデルのあてはまりが良く、無駄なパラメータが少ない良いモデルであると判断されます（ペナルティ項 $2k$ が過学習を防ぐ）。

2.  **各モデルのAIC計算**
    - **モデル1のAIC**:
      $$\text{AIC}_1 = -2 \times (-150) + 2 \times 3 = 300 + 6 = 306$$
    - **モデル2のAIC**:
      $$\text{AIC}_2 = -2 \times (-147) + 2 \times 5 = 294 + 10 = 304$$

3.  **モデル選択の判定**
    $\text{AIC}_2 = 304 < \text{AIC}_1 = 306$ であるため、パラメータ数は増えたものの、それ以上に対数尤度の改善幅が大きいため、**モデル2のほうが適している**と判断されます。

    したがって、正解は **1** です。

## 16. シミュレーション・計算機統計分野
### 📌 キーワード：ブートストラップ法 (Ch 32)

#### 【問題】
サイズ $n$ の元のデータサンプル $\mathbf{X} = (X_1, X_2, \dots, X_n)$ から、ブートストラップ標本（Bootstrap Sample） $\mathbf{X}^* = (X_1^*, \dots, X_n^*)$ を生成するリサンプリングのルールとして正しい記述を以下の選択肢から選べ。

1.  元のサンプルから重複を許さずに（非復元抽出）、サイズ $n$ のサンプルを抽出する。
2.  元のサンプルから重複を許して（復元抽出）、サイズ $n$ のサンプルを抽出する。
3.  元のサンプルから重複を許さずに（非復元抽出）、サイズ $n/2$ のサンプルを抽出する。
4.  標準正規分布 $N(0, 1)$ からランダムにサイズ $n$ の疑似乱数を生成する。
5.  元のサンプルの各データに、小さなランダムノイズを加えたデータをサイズ $n$ 分生成する。

#### 【解答】
**2.  元のサンプルから重複を許して（復元抽出）、サイズ $n$ のサンプルを抽出する。**

#### 【解説】
1.  **ブートストラップ標本の生成**
    ブートストラップ法（ノンパラメトリック・ブートストラップ）では、観測された標本自体を「経験分布（母集団の近似）」とみなし、そこから**復元抽出（重複を許す）**によって**元と全く同じサイズ $n$** の標本を何度も再抽出します。
    非復元で $n$ 個抽出すると元と全く同一のサンプルしか得られないため、復元抽出であることが本質的です。

    したがって、正しい記述は **2** となります。
