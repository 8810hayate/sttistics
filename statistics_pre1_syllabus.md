# 統計検定準1級 出題範囲・学習ガイド

統計検定準1級は、大学で学ぶ応用的な統計学の諸手法を実社会の問題に応用できる能力を評価する試験です。出題範囲は非常に広範で、日本統計学会公式認定の『統計学実践ワークブック』に準拠しています。

---

## 1. 試験概要

| 項目 | 内容 |
| :--- | :--- |
| **試験方式** | CBT（Computer Based Testing）方式（随時受験可） |
| **試験時間** | 90分 |
| **問題数** | 約20〜30問（大問形式、選択式・数値入力式） |
| **合格基準** | 正答率 約60%〜70%以上（難易度により変動） |
| **持ち込み** | 電卓（関数電卓は不可） |

> [!NOTE]
> 2021年以降、CBT方式に完全移行しました。計算量が多く迅速かつ正確な処理能力が求められます。各章の例題や問が形を変えてそのまま出題される傾向があります。

---

## 2. 分野別・詳細シラバス（全32章）

『統計学実践ワークブック』の全32章を6つの主要分野に整理し、それぞれの章の重要キーワードをまとめています。

### I. 確率と確率分布（第1章〜第7章）
確率論の基礎から、連続・離散の主要分布、漸近理論（極限定理）まで。2級の知識をベースに、より高度な多変数変換やモーメント母関数の計算が加わります。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第1章** | **事象と確率** | 事象、確率の公理、条件付き確率、**ベイズの定理**、独立性 |
| **第2章** | **確率分布と母関数** | 累積分布関数、確率（密度）関数、**確率母関数**、**モーメント母関数** |
| **第3章** | **分布の特性値** | 期待値、分散、共分散、相関係数、**歪度・尖度**、条件付き期待値と分散 |
| **第4章** | **変数変換** | **ヤコビアン（多変数変換）**、たたみ込み、商・積の分布、**順序統計量** |
| **第5章** | **離散型分布** | ベルヌーイ、二項、ポアソン、幾何、**負の二項**、超幾何、**多項分布** |
| **第6章** | **連続型分布と標本分布** | 正規、対数正規、一様、指数、**ガンマ**、**ベータ**、コーシー、多変量正規、$\chi^2$、t、F分布 |
| **第7章** | **極限定理，漸近理論** | 大数の法則、中心極限定理、**漸近正規性**、**デルタ法**、スラスキーの定理 |

> [!IMPORTANT]
> - **ヤコビアンを用いた変数変換**は計算問題として超頻出です。
> - **負の二項分布**、**多項分布**、**ガンマ分布**、**ベータ分布**の定義式と期待値・分散は暗記必須です。

---

### II. 統計的推測の基礎（第8章〜第13章）
点推定・区間推定の数学的裏付け、仮説検定の導出理論、および正規分布に依存しないノンパラメトリック検定をカバーします。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第8章** | **統計的推定の基礎** | 不偏性、一致性、**有効性**、**フィッシャー情報量**、**十分統計量**、最尤法、モーメント法 |
| **第9章** | **区間推定** | 枢軸変数、信頼区間、信頼係数、**被覆確率** |
| **第10章** | **検定の基礎と検定法の導出** | 第1種・第2種の過誤、**検出力（1-β）**、**ネイマン・ピアソンの基本定理**、**尤度比検定** |
| **第11章** | **正規分布に関する検定** | 1標本/2標本t検定、F検定、等分散の検定、ウェルチの検定 |
| **第12章** | **一般の分布に関する検定法** | カイ二乗適合度検定、独立性の検定、尤度比適合度検定 |
| **第13章** | **ノンパラメトリック法** | 符号検定、**ウィルコクソンの順位和検定**、**符号付き順位検定**、クラスカル・ウォリス |

> [!IMPORTANT]
> - **十分統計量の判定**（因子分解定理）と**フィッシャー情報量の計算**（クラメール・ラオの下限）は、ほぼ毎回出題される最重要トピックです。

---

### III. 回帰分析と分散分析（第16章〜第20章）
単回帰・重回帰だけでなく、残差のチェック（診断法）、ロジスティック回帰などの質的回帰、スプラインなどの非線形、分散分析や実験計画法まで含みます。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第16章** | **重回帰分析** | 最小二乗法、ガウス・マルコフの定理、多重共線性（VIF）、自由度調整済み決定係数 |
| **第17章** | **回帰診断法** | 残差プロット、Q-Qプロット、**ハット行列（レバレッジ）**、**クックの距離**、DW比 |
| **第18章** | **質的回帰** | **ロジスティック回帰**、プロビット回帰、**オッズ比**、多項ロジスティック回帰 |
| **第19章** | **回帰分析その他** | **一般化線形モデル（GLM）**、リンク関数、**リッジ回帰・Lasso回帰（正則化）**、スプライン |
| **第20章** | **分散分析と実験計画法** | 一元/二元配置分散分析、交互作用、**フィッシャーの3原則**、乱塊法、直交表 |

---

### IV. 多変量解析（第22章〜第26章）
多次元データを要約・分類・分析する手法群。数式の意味（特に固有値問題や距離の概念）と結果の解釈が出題されます。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第22章** | **主成分分析** | 固有値・固有ベクトル、寄与率、累積寄与率、主成分負荷量、主成分得点 |
| **第23章** | **判別分析** | **フィッシャーの線形判別関数**、**マハラノビス汎距離**、誤判別確率、マハラノビス距離 |
| **第24章** | **クラスター分析** | 階層的（**最短・最長・群平均・ウォード法**）、非階層的（**k-means**）、コフェネット相関 |
| **第25章** | **因子分析・グラフィカルモデル** | 共通因子、独自因子、因子負荷量、因子回転（直交・斜交）、**条件付き独立**、無向/有向グラフ |
| **第26章** | **その他の多変量解析手法** | MDS（多次元尺度構成法）、**数量化理論（I, II, III, IV類）**、**サポートベクトルマシン（SVM）**、決定木 |

---

### V. 確率過程と時系列（第14, 15, 27章）
時間の経過とともに変化する確率的な現象を扱う、準1級特有の難所です。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第14章** | **マルコフ連鎖** | 推移確率行列、定常分布、**エルゴード性**、吸収状態、吸収確率 |
| **第15章** | **確率過程の基礎** | ランダムウォーク、**ポアソン過程**、ブラウン運動、出生死滅過程 |
| **第27章** | **時系列解析** | 定常性、自己相関、偏自己相関、**AR、MA、ARMA、ARIMAモデル**、**ARCH/GARCH** |

---

### VI. 実用的・応用統計手法（第21, 28章〜第32章）
実際のデータ解析の現場で必要となる、標本抽出、不完全データ処理、ベイズ統計、計算機シミュレーションなど。

| 章 | タイトル | 重要キーワード・概念 |
| :--- | :--- | :--- |
| **第21章** | **標本調査法** | 単純無作為、系統、**層化**、**集落**、多段抽出、有限母集団修正 |
| **第28章** | **分割表** | 対数線形モデル、オッズ比、コホート・症例対照研究、マクネマー検定 |
| **第29章** | **不完全データの統計処理** | **MCAR/MAR/MNAR（欠損メカニズム）**、代入法、**EMアルゴリズム** |
| **第30章** | **モデル選択** | **AIC（赤池情報量基準）**、**BIC（ベイズ情報量基準）**、クロスバリデーション（CV） |
| **第31章** | **ベイズ法** | 事前分布（**共役事前分布**）、事後分布、MCMC（ギブス、MH法） |
| **第32章** | **シミュレーション** | モンテカルロ積分、**重要度サンプリング**、**ブートストラップ法**、ジャックナイフ法 |

---

## 3. 合格に向けた学習戦略と優先度

準1級は範囲が非常に広いため、すべての分野を完璧にするのではなく、**配点割合が高く得点源にしやすい分野**から優先的に攻略するのが鉄則です。

```
【優先度 A】確率・推定・検定の理論（第1章〜第13章）
  └ 全体の4〜5割近くを占める土台。計算問題も理論問題もここから多く出ます。特に十分統計量、最尤推定、ヤコビアン、ベイズの定理は外せません。

【優先度 B】回帰分析と多変量解析（第16章〜第19章, 第22章〜第23章）
  └ 重回帰、ロジスティック回帰、主成分分析、判別分析は、数式の行列表現と結果の読み取りが頻出です。

【優先度 C】時系列解析と分割表・モデル選択（第27章, 第28章, 第30章）
  └ 実社会でのデータ解析でよく使われるため、出題頻度が高めです。
```
