分散分析

Analysis of Variance（ANOVA）

→ キーまたはクリックで進む

01

分散分析とは？

🎯

3つ以上のグループの母平均に差があるかを検定する手法

例

「学習法 A・B・C でテストの点数に差があるか？」

💡 平均の比較なのに「分散」分析？

→ データ全体のばらつき（分散）を要因ごとに分解して平均の差を検定するため

02

なぜ t検定の繰り返しではダメ？

ペアごとに t検定を繰り返すと第1種の過誤（偽陽性）が膨らむ

グループ数 ペア数 誤る確率 (α=0.05)
3 3 約 14%
5 10 約 40%
10 45 約 90%

✅ 分散分析なら 1回の検定 で全体の差を判定 → 過誤を制御できる

03

基本的な考え方：分散の分解

全体の変動
SST
=
群間変動
SSB
+
群内変動
SSW

群間変動（SSB）

Sum of Squares Between

各群の平均と全体平均の差

→ 要因の効果

群内変動（SSW）

Sum of Squares Within

各群内でのばらつき

→ 誤差（偶然）

群間変動が群内変動に比べて 十分に大きい → 「有意な差がある」

04

理論① 統計モデル

xij = μ + αi + εij
xij 第 i 群の j 番目のデータ
μ 全体の平均（共通部分）
αi 第 i 群の効果（その群特有のずれ）
εij 誤差項 〜 N(0, σ²)　独立・等分散

帰無仮説 H₀ : α₁ = α₂ = … = αk = 0
（すべての群の効果がゼロ ＝ 群間に差がない）

📝 このモデルは「データ ＝ 全体平均 ＋ 群の効果 ＋ 偶然の誤差」という構造を仮定している

05

理論② MSB と MSW は何を推定するか

これが分散分析の理論の核心です

MSW（群内平均平方）

常に成り立つ

E[MSW] = σ²

群の平均が何であっても、群内のばらつきは誤差 ε だけで決まる

MSB（群間平均平方）

H₀ が正しいかどうかで変わる

E[MSB] = σ² + 群間差

H₀ のもとでは σ² のみ
H₁ のもとでは σ² ＋ α の効果

💡

同じ σ² を2つの異なる方法で推定し、その比をとる——これが分散分析のエッセンス

06

理論③ なぜ F値で判定できるのか

H₀ が正しいとき
MSB の期待値σ²
MSW の期待値σ²
F = MSB / MSW≈ 1
分子と分母が同じものを推定
→ 比は 1 付近に
H₁ が正しいとき
MSB の期待値σ² + 群間差
MSW の期待値σ²
F = MSB / MSW≫ 1
分子だけが膨らむ
→ 比が 1 より大きくなる

F が「1 からどれだけ離れたか」を F分布で評価 → 棄却域や p値を計算できる

07

理論④ なぜ F分布に従うのか

正規分布の仮定のもとで、以下が成り立ちます

①

SSW / σ² 〜 χ²(N − k)

各群の偏差二乗和の合計

②

H₀ のもとで SSB / σ² 〜 χ²(k − 1)

群平均のばらつきが偶然だけで生じる場合

③

SSB と SSW は独立

射影の直交性から証明できる

F = χ²(k−1) / (k−1)χ²(N−k) / (N−k) 〜 F(k−1, N−k)

📝 F分布の定義 =「独立な2つの χ² 変数を各自由度で割った比」にぴったり一致する

08

用語一覧

略称 英語 日本語

SSB Sum of Squares Between 群間平方和
SSW Sum of Squares Within 群内平方和

MSW Mean Square Within 群内平均平方

📝 教科書により SSB → SSA、SSW → SSE (Error) と書くこともある

09

F 検定統計量

F = MSBMSW = SSB / (k − 1)SSW / (N − k)
k 群の数
N 全データ数
k − 1 群間の自由度
N − k 群内の自由度

F値が大きい → 群間の差が大きい → 帰無仮説を棄却

F値は F分布（自由度 k−1, N−k）に従う

10

計算例：データの確認

問題

3種類の学習法（A, B, C）でテストの点数に差があるか？（有意水準5%）

学習法A n=3

607080
平均 = 70

学習法B n=5

7080607565
平均 = 70

学習法C n=4

80908595
平均 = 87.5
全体平均: x̄ = 910 / 12 ≈ 75.83

⚠️ 各群のデータ数が異なる（3, 5, 4）→ SSBの重みづけに注意

11

計算① 群間変動（SSB）

SSB = Σ ni (x̄i − x̄)2
A: 3 × (70 − 75.83)² = 3 × 34.03 = 102.08
B: 5 × (70 − 75.83)² = 5 × 34.03 = 170.14
C: 4 × (87.5 − 75.83)² = 4 × 136.11 = 544.44
SSB = 102.08 + 170.14 + 544.44 = 816.67
💡

A と B は同じ平均（70）でも、データ数の重み（3 vs 5）で SSB への寄与が異なる

12

計算② 群内変動（SSW）

SSW = Σ Σ (xij − x̄i)2
A (60−70)² + (70−70)² + (80−70)² = 200
B (70−70)² + (80−70)² + (60−70)² + (75−70)² + (65−70)² = 250
C (80−87.5)² + (90−87.5)² + (85−87.5)² + (95−87.5)² = 125
SSW = 200 + 250 + 125 = 575

各データと その群の平均 との差の二乗和（全体平均ではない点に注意）

15

検定の流れ（まとめ）

1
仮説を立てる

H₀: μ₁ = μ₂ = … = μk

↓
2
各群の平均と全体平均を計算
↓
3
SSB・SSW を求める

SST = SSB + SSW で検算

↓
4
分散分析表を作成し F値を計算

F = MSB / MSW

↓
5
F分布表と比較し判定

F > Fα なら棄却

16

分散分析の前提条件

1

独立性

各観測値が互いに独立であること

2

正規性

各群のデータが正規分布に従うこと

3

等分散性

各群の母分散が等しいこと
（Levene検定等で確認）

まとめ

分散分析（ANOVA）

✅ 3群以上の平均の差を1回の検定で判定

✅ SST = SSB + SSW に分解

✅ MSB と MSW が同じ σ² を推定するかで判定

✅ F = MSB / MSW → F分布で検定

✅ 有意ならば多重比較へ
