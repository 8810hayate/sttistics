📈 回帰分析とは？

「ある変数（目的変数）を別の変数（説明変数）で予測・説明する手法」
💡 例：勉強時間 (x) からテスト点数 (y) を予測

2つのタイプ

📊 単回帰分析

説明変数が 1つ のモデル

y = β₀ + β₁x + ε

📊 重回帰分析

説明変数が 2つ以上 のモデル

y = β₀ + β₁x₁ + β₂x₂ + … + ε

02 / 15

📐 単回帰モデル

単回帰モデル yi = β₀ + β₁xi + εi
  • yi 目的変数（被説明変数、応答変数）
  • xi 説明変数（独立変数、予測変数）
  • β₀ 切片（x = 0 のときの y の期待値）
  • β₁ 回帰係数：x が1増えたときの y の変化量
  • εi 誤差項 〜 N(0, σ²)　（正規分布に従う）
🎯 目標 = データから β₀ と β₁ を推定すること

✏️ 導出① β₀ の偏微分

1 ∂S/∂β₀ = −2Σ(yi − β₀ − β₁xi) = 0
2 Σyi − nβ₀ − β₁Σxi = 0
3 n で割る：ȳ − β₀ − β₁x̄ = 0
∴ β̂₀ = ȳ − β̂₁x̄
💡 回帰直線は必ず (x̄, ȳ) を通る
05 / 15

✏️ 導出② β₁ の偏微分

1 ∂S/∂β₁ = −2Σxi(yi − β₀ − β₁xi) = 0
2 Σxiyi − β₀Σxi − β₁Σxi² = 0
3 β₀ = ȳ − β₁x̄ を代入して整理
∴ Σxiyi − nx̄ȳ = β₁(Σxi² − nx̄²)
📝 左辺 = Sxy（偏差積和）、右辺の括弧 = Sxx（偏差平方和）
06 / 15

🏁 最小二乗推定量

回帰係数 β̂₁ = SxySxx = Σ(xi − x̄)(yi − ȳ)Σ(xi − x̄)²
切片 β̂₀ = ȳ − β̂₁x̄
  • Sxy x と y の偏差積和（共変動）
  • Sxx x の偏差平方和（x のばらつき）
💡 β̂₁ = 「x と y の共変動」÷「x の変動」

🧪 回帰係数の検定

帰無仮説 H₀

β₁ = 0（線形関係なし）

対立仮説 H₁

β₁ ≠ 0（線形関係あり）

検定統計量 t = β̂₁SE(β̂₁)
  • SE(β̂₁) = √(σ̂² / Sxx)　β̂₁ の標準誤差
  • σ̂² = SSE / (n − 2)　誤差分散の推定値
  • 自由度 n − 2（パラメータ β₀, β₁ の2つを推定）
📌 |t| > tα/2(n−2) なら「有意な線形関係がある」と判断

⚖️ 自由度調整済み R²

変数を増やすと R² は必ず増加 → 変数の数でペナルティをかける

自由度調整済み決定係数 R̄² = 1 − SSE / (n − p − 1)SST / (n − 1)

📊 R²（通常）

  • 変数追加で必ず↑
  • 過学習を見抜けない

📊 調整済み R²

  • 不要な変数追加で↓
  • モデル選択に有用

⚠️ 重回帰の注意点

1

多重共線性

説明変数同士の相関が高いと推定が不安定になる。VIF（分散拡大因子）で確認する。

2

変数選択

AIC・BIC 等の情報量基準で不要な変数を除き、最適なモデルを選ぶ。

3

ダミー変数

カテゴリ変数を 0/1 で数値化して回帰モデルに含める。

13 / 15

🔍 残差分析

モデルの妥当性を残差のパターンから確認する

1

等分散性

残差のばらつきが一定か確認（残差 vs 予測値プロット）

2

正規性

残差が正規分布に従うか確認（Q-Qプロット）

3

独立性

残差に系列相関がないか確認（ダービン・ワトソン統計量）

4

線形性

x と y の関係が線形か確認（残差に曲線パターンがないこと）

🗺️ 回帰分析の流れ（まとめ）

1 散布図で関係を確認
↓
2 最小二乗法で β を推定
↓
3 R² で当てはまりを評価
↓
4 t検定 / F検定で有意性を検定
↓
5 残差分析でモデルの妥当性を確認
まとめ

回帰分析

  • ✅ 最小二乗法で β₀, β₁ を推定
  • ✅ β̂₁ = Sxy / Sxx
  • ✅ R² = 1 − SSE/SST で当てはまり評価
  • ✅ t検定で回帰係数の有意性を判定
  • ✅ 重回帰では調整済み R² を使う
